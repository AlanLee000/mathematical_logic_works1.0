## 等价与应用理论MEA

本系统是一个基于带等词的一阶谓词逻辑的形式理论。

### **1. 形式语言 $MEA$**

**1.1. 符号表 (Alphabet)**

1.  **变量 (Variables):** 一个可数无限集 $V = \{v_0, v_1, v_2, \dots\}$。为方便起见，我们使用 $x, y, z, \dots$ 作为其元变量。
2.  **常量 (Constants):** 一个常量符号: `1`。
3.  **函数符号 (Function Symbols):**
    *   `equiv`: 二元函数符号。
    *   `·`: 二元函数符号。
4.  **谓词符号 (Predicate Symbols):**
    *   ` = `: 二元谓词符号。
5.  **逻辑联结词 (Logical Connectives):** `¬` (否定), `→` (蕴含)。
6.  **量词 (Quantifiers):** `∀` (全称量词)。
7.  **辅助符号 (Auxiliary Symbols):** `(`, `)`。

**1.2. 项 (Terms)**

$MEA$ 的项集合是满足以下条件的最小集合:
1.  任何变量 $v \in V$ 都是一个项。
2.  常量 `1` 是一个项。
3.  若 $t_1$ 和 $t_2$ 是项，则 `equiv(t₁, t₂)` 是一个项。
4.  若 $t_1$ 和 $t_2$ 是项，则 `·(t₁, t₂)` 是一个项。

**1.3. 合式公式 (Well-Formed Formulas - WFFs)**

$MEA$ 的合式公式集合是满足以下条件的最小集合:
1.  若 $t_1$ 和 $t_2$ 是项，则 `t₁ = t₂` 是一个原子公式，也是一个合式公式。
2.  若 $\phi$ 是一个合式公式，则 `(¬φ)` 是一个合式公式。
3.  若 $\phi$ 和 $\psi$ 是合式公式，则 `(φ → ψ)` 是一个合式公式。
4.  若 $\phi$ 是一个合式公式且 $x$ 是一个变量，则 `(∀x φ)` 是一个合式公式。

**1.4. 辅助定义**

为简洁起见，我们引入以下元语言缩写：
*   `(φ ∧ ψ)` 定义为 `¬(φ → ¬ψ)`
*   `(φ ∨ ψ)` 定义为 `(¬φ → ψ)`
*   `(φ ↔ ψ)` 定义为 `((φ → ψ) ∧ (ψ → φ))`
*   `(∃x φ)` 定义为 `¬(∀x ¬φ)`
*   `t₁ ≠ t₂` 定义为 `¬(t₁ = t₂)`

### **2. 公理系统 $A_{EA}$**

公理系统由逻辑公理、推理规则和非逻辑公理组成。

**2.1. 逻辑公理**

对于任意合式公式 $\phi, \psi, \chi$：

*   **A1:** $(\phi \to (\psi \to \phi))$
*   **A2:** $((\phi \to (\psi \to \chi)) \to ((\phi \to \psi) \to (\phi \to \chi)))$
*   **A3:** $((¬\phi \to ¬\psi) \to (\psi \to \phi))$
*   **A4:** $(\forall x \phi(x) \to \phi(t))$, 其中项 $t$ 在 $\phi(x)$ 中对 $x$ 是自由可代入的。
*   **A5:** $(\forall x (\phi \to \psi) \to (\phi \to \forall x \psi))$, 其中 $\phi$ 中不含自由出现的变量 $x$。

**2.2. 等词公理 (Axioms of Equality)**

对于任意变量 $x, y$ 和项 $t_1, \dots, t_n, s_1, \dots, s_n$：

*   **A6 (自反性):** $\forall x (x = x)$
*   **A7 (替换公理模式):**
    *   对于任意 $n$ 元函数符号 $f$ (在 $MEA$ 中为 `equiv` 和 `·`):
        $\forall x_1 \dots \forall x_n \forall y_1 \dots \forall y_n ( (x_1=y_1 \land \dots \land x_n=y_n) \to f(x_1, \dots, x_n) = f(y_1, \dots, y_n) )$
    *   对于任意 $n$ 元谓词符号 $P$ (在 $MEA$ 中为 ` = `)
        $\forall x_1 \dots \forall x_n \forall y_1 \dots \forall y_n ( (x_1=y_1 \land \dots \land x_n=y_n) \to (P(x_1, \dots, x_n) \to P(y_1, \dots, y_n)) )$

**2.3. 推理规则 (Rules of Inference)**

1.  **分离规则 (Modus Ponens):** 从 $\phi$ 和 $(\phi \to \psi)$，可推出 $\psi$。
2.  **全称概括 (Generalization):** 从 $\phi$，可推出 $(\forall x \phi)$。

**2.4. 非逻辑公理 (Non-Logical Axioms)**

以下公理定义了理论 $MEA$ 的特定性质。

*   **EA1 (等价定义):**
    $\forall x \forall y ( \text{equiv}(x, y) = 1 \leftrightarrow x = y )$

*   **EA2 (等价交换律):**
    $\forall x \forall y ( \text{equiv}(x, y) = \text{equiv}(y, x) )$

*   **EA3 (左逆元):**
    $\forall x \exists y (\cdot(y, x) = 1)$

*   **EA4 (结合律):**
    $\forall x, y, z (\cdot(x, \cdot(y, z)) = \cdot(\cdot(x, y), z))$

*   **EA5 (混合结合律):**
    $\forall x \forall y \forall z ( \text{equiv}(\cdot(x, y), z) = \cdot(x, \text{equiv}(y, z)) )$

### 3. 模型论语义

#### **3.1. 模型的构建**

为了找到一个“退化”但“非平凡”的模型，我们应该寻找最简单的可能结构。“非平凡”意味着论域（domain）的基数必须大于1。因此，我们从基数为2的论域开始。

我们定义模型 $\mathcal{M}$ 如下：

$\mathcal{M} = \langle \mathcal{D}, 1^\mathcal{M}, \text{equiv}^\mathcal{M}, \cdot^\mathcal{M}, =^\mathcal{M} \rangle$

1.  **论域 (Domain):** $\mathcal{D} = \{T, F\}$。这是一个包含两个不同元素的集合。

2.  **常量解释:**
    *   $1^\mathcal{M} = T$。我们将元素 $T$ 解释为常量 `1`。

3.  **谓词解释:**
    *   $=^\mathcal{M}$ 是 $\mathcal{D}$ 上的标准恒等关系。即，对于任意 $d_1, d_2 \in \mathcal{D}$，$d_1 =^\mathcal{M} d_2$ 当且仅当 $d_1$ 和 $d_2$ 是同一个元素。

4.  **函数解释:**

    *   **`equiv` 的解释 ($\text{equiv}^\mathcal{M}$):**
        该函数的解释受到公理 **EA1** `(equiv(x, y) = 1 ↔ x = y)` 的严格约束。
        *   如果 $x=y$，则 $\text{equiv}(x, y)$ 必须被解释为 $1^\mathcal{M} = T$。
        *   如果 $x \neq y$，则 $\text{equiv}(x, y)$ 必须被解释为不等于 $1^\mathcal{M}$，在我们的论域中即为 $F$。
        因此，$\text{equiv}^\mathcal{M}$ 必须是逻辑上的 **等价/同或 (XNOR)** 运算：
        *   $\text{equiv}^\mathcal{M}(T, T) = T$
        *   $\text{equiv}^\mathcal{M}(F, F) = T$
        *   $\text{equiv}^\mathcal{M}(T, F) = F$
        *   $\text{equiv}^\mathcal{M}(F, T) = F$

    *   **`·` 的解释 ($\cdot^\mathcal{M}$):**
        该函数的解释需要同时满足结合律 (EA4)，左逆元 (EA3) 和关键的混合结合律 (EA5)。一个绝佳的候选者是 **异或 (XOR)** 运算，它在集合 $\{T, F\}$ （通常对应于 $\{0, 1\}$）上构成一个阿贝尔群。
        我们定义 $\cdot^\mathcal{M}$ 如下：
        *   $T \cdot^\mathcal{M} T = F$
        *   $T \cdot^\mathcal{M} F = T$
        *   $F \cdot^\mathcal{M} T = T$
        *   $F \cdot^\mathcal{M} F = F$
        
        *注：如果我们把 $T$ 映射到 1， $F$ 映射到 0，那么 $\cdot^\mathcal{M}$ 就是模2加法 ($+$)，而 $equiv^\mathcal{M}$ 就是 $1 - (x+y) \pmod 2$。但为了避免混淆，我们全程使用 $T/F$。*

这个模型是 **非平凡的**，因为其论域有两个元素。它是 **退化的**，因为这是满足非平凡条件的最小可能论域。

#### **3.2. 模型满足公理**

现在，我们必须证明我们构建的模型 $\mathcal{M}$ 满足 $A_{EA}$ 的所有公理。这意味着对于任何变量赋值函数 $s: V \to \mathcal{D}$，所有公理在 $(\mathcal{M}, s)$ 下均为真。

##### **3.2.1. 逻辑公理和等词公理**

*   **A1-A5 (逻辑公理):** 这些是任何标准的一阶逻辑模型都必须满足的重言式和量化模式。我们的模型 $\mathcal{M}$ 是一个标准的 Tarski-style 模型，因此这些公理自然成立。
*   **A6 (等词自反性):** $\forall x (x = x)$。在 $\mathcal{M}$ 中，$T=T$ 和 $F=F$ 都为真。公理成立。
*   **A7 (替换公理):** 我们的函数 $\text{equiv}^\mathcal{M}$ 和 $\cdot^\mathcal{M}$ 都是良定义的映射。例如，如果 $x_1=y_1$ 且 $x_2=y_2$，那么 $x_1$ 和 $y_1$ 是同一个元素，$x_2$ 和 $y_2$ 也是同一个元素，因此 $\cdot^\mathcal{M}(x_1, x_2)$ 和 $\cdot^\mathcal{M}(y_1, y_2)$ 必然得出相同的结果。这对两个函数都成立，因此替换公理成立。

##### **3.2.2. 非逻辑公理**

这是证明的核心。我们需要逐一验证 EA1 到 EA5。

*   **EA1 (等价定义):** $\forall x \forall y ( \text{equiv}(x, y) = 1 \leftrightarrow x = y )$
    *   在 $\mathcal{M}$ 中，这等价于 $\text{equiv}^\mathcal{M}(d_1, d_2) = T \iff d_1 = d_2$ 对于所有 $d_1, d_2 \in \{T, F\}$。
    *   **情况 1 ($d_1 = d_2$):**
        *   $\text{equiv}^\mathcal{M}(T, T) = T$。 $T=T$ 为真，$\leftrightarrow$ 成立。
        *   $\text{equiv}^\mathcal{M}(F, F) = T$。 $T=T$ 为真，$\leftrightarrow$ 成立。
    *   **情况 2 ($d_1 \neq d_2$):**
        *   $\text{equiv}^\mathcal{M}(T, F) = F$。 $F=T$ 为假，$\leftrightarrow$ 成立。
        *   $\text{equiv}^\mathcal{M}(F, T) = F$。 $F=T$ 为假，$\leftrightarrow$ 成立。
    *   **结论:** 公理 **EA1** 成立。

*   **EA2 (等价交换律):** $\forall x \forall y ( \text{equiv}(x, y) = \text{equiv}(y, x) )$
    *   我们的 $\text{equiv}^\mathcal{M}$ 定义是对称的：$\text{equiv}^\mathcal{M}(T, F) = F = \text{equiv}^\mathcal{M}(F, T)$。其他情况（$x=y$）显然也对称。
    *   **结论:** 公理 **EA2** 成立。

*   **EA3 (左逆元):** $\forall x \exists y (\cdot(y, x) = 1)$
    *   在 $\mathcal{M}$ 中，这等价于对于每个 $d_1 \in \{T, F\}$，存在一个 $d_2 \in \{T, F\}$ 使得 $d_2 \cdot^\mathcal{M} d_1 = T$。
    *   **当 $x=T$:** 我们需要找到 $y$ 使得 $y \cdot^\mathcal{M} T = T$。根据我们的定义，$F \cdot^\mathcal{M} T = T$。所以 $y=F$ 存在。
    *   **当 $x=F$:** 我们需要找到 $y$ 使得 $y \cdot^\mathcal{M} F = T$。根据我们的定义，$T \cdot^\mathcal{M} F = T$。所以 $y=T$ 存在。
    *   **结论:** 公理 **EA3** 成立。

*   **EA4 (结合律):** $\forall x, y, z (\cdot(x, \cdot(y, z)) = \cdot(\cdot(x, y), z))$
    *   我们的运算 $\cdot^\mathcal{M}$ 是异或 (XOR)。众所周知，XOR 运算是满足结合律的。例如，我们可以检验一种情况：
        *   $x=T, y=T, z=F$:
            *   左边: $T \cdot^\mathcal{M} (T \cdot^\mathcal{M} F) = T \cdot^\mathcal{M} T = F$。
            *   右边: $(T \cdot^\mathcal{M} T) \cdot^\mathcal{M} F = F \cdot^\mathcal{M} F = F$。
        *   对所有8种组合进行检验都会得到相同的结果。
    *   **结论:** 公理 **EA4** 成立。

*   **EA5 (混合结合律):** $\forall x \forall y \forall z ( \text{equiv}(\cdot(x, y), z) = \cdot(x, \text{equiv}(y, z)) )$
    *   这是最关键的检验。我们需要对所有 $x, y, z \in \{T, F\}$ 的8种组合进行验证。

    1.  **x=T, y=T, z=T:**
        *   左边: $\text{equiv}^\mathcal{M}(T \cdot^\mathcal{M} T, T) = \text{equiv}^\mathcal{M}(F, T) = F$
        *   右边: $T \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(T, T) = T \cdot^\mathcal{M} T = F$。 **(匹配)**
    2.  **x=T, y=T, z=F:**
        *   左边: $\text{equiv}^\mathcal{M}(T \cdot^\mathcal{M} T, F) = \text{equiv}^\mathcal{M}(F, F) = T$
        *   右边: $T \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(T, F) = T \cdot^\mathcal{M} F = T$。 **(匹配)**
    3.  **x=T, y=F, z=T:**
        *   左边: $\text{equiv}^\mathcal{M}(T \cdot^\mathcal{M} F, T) = \text{equiv}^\mathcal{M}(T, T) = T$
        *   右边: $T \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(F, T) = T \cdot^\mathcal{M} F = T$。 **(匹配)**
    4.  **x=T, y=F, z=F:**
        *   左边: $\text{equiv}^\mathcal{M}(T \cdot^\mathcal{M} F, F) = \text{equiv}^\mathcal{M}(T, F) = F$
        *   右边: $T \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(F, F) = T \cdot^\mathcal{M} T = F$。 **(匹配)**
    5.  **x=F, y=T, z=T:**
        *   左边: $\text{equiv}^\mathcal{M}(F \cdot^\mathcal{M} T, T) = \text{equiv}^\mathcal{M}(T, T) = T$
        *   右边: $F \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(T, T) = F \cdot^\mathcal{M} T = T$。 **(匹配)**
    6.  **x=F, y=T, z=F:**
        *   左边: $\text{equiv}^\mathcal{M}(F \cdot^\mathcal{M} T, F) = \text{equiv}^\mathcal{M}(T, F) = F$
        *   右边: $F \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(T, F) = F \cdot^\mathcal{M} F = F$。 **(匹配)**
    7.  **x=F, y=F, z=T:**
        *   左边: $\text{equiv}^\mathcal{M}(F \cdot^\mathcal{M} F, T) = \text{equiv}^\mathcal{M}(F, T) = F$
        *   右边: $F \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(F, T) = F \cdot^\mathcal{M} F = F$。 **(匹配)**
    8.  **x=F, y=F, z=F:**
        *   左边: $\text{equiv}^\mathcal{M}(F \cdot^\mathcal{M} F, F) = \text{equiv}^\mathcal{M}(F, F) = T$
        *   右边: $F \cdot^\mathcal{M} \text{equiv}^\mathcal{M}(F, F) = F \cdot^\mathcal{M} T = T$。 **(匹配)**

    *   所有情况均匹配。
    *   **结论:** 公理 **EA5** 成立。

由于模型 $\mathcal{M}$ 满足 MEA 理论的所有公理，我们已经证明了该模型的 **可靠性 (Soundness)**。

#### **3.3. 非平凡一致性证明 (Proof of Non-trivial Consistency)**

一致性（Consistency）是指一个理论无法推导出矛盾，即不存在某个公式 $\phi$ 使得理论能同时证明 $\phi$ 和 $\neg\phi$。

**根据一阶逻辑的可靠性定理 (Soundness Theorem) ，当一个理论有一个模型时，这个理论是一致的。**

1.  **存在模型:** 我们在上面已经成功地构建了一个模型 $\mathcal{M}$，并且证明了 MEA 理论的所有公理在该模型中都为真。

2.  **一致性:** 既然 MEA 理论有一个模型 $\mathcal{M}$，那么它必定是 **一致的**。
    *   **论证:** 假设 MEA 是不一致的，那么它可以推导出一条矛盾，例如 `1 ≠ 1`（即 `¬(1=1)`）。根据一阶逻辑的可靠性，任何可以从公理中推导出的定理都必须在所有模型中为真。因此，`1 ≠ 1` 必须在我们的模型 $\mathcal{M}$ 中为真。
    *   然而，在 $\mathcal{M}$ 中，`1` 被解释为 $T$，` = ` 被解释为恒等关系。所以 `1=1` 在 $\mathcal{M}$ 中是 $T=T$，这是真的。因此 `¬(1=1)` 在 $\mathcal{M}$ 中是假的。
    *   这导致了矛盾：如果 MEA 不一致，`1 ≠ 1` 在 $\mathcal{M}$ 中必须为真，但我们已证明它在 $\mathcal{M}$ 中为假。因此，我们的初始假设（MEA 不一致）是错误的。

3.  **非平凡性:** 一致性分为平凡一致和非平凡一致。
    *   **平凡一致 (Trivial Consistency):** 理论是一致的，但它的所有模型都是平凡的（即论域只有一个元素）。
    *   **非平凡一致 (Non-trivial Consistency):** 理论存在至少一个非平凡模型。

    由于我们构建的模型 $\mathcal{M}$ 的论域 $\mathcal{D} = \{T, F\}$ 的基数为2（大于1），所以 $\mathcal{M}$ 是一个非平凡模型。

    **因此，我们证明了 MEA 理论是“非平凡一致的”。**

### **4. 理论推论与模型结构**

本节将从公理系统 $A_{EA}$ 出发，推导其任意模型的代数结构必须遵循的若干核心性质。

**前提:** 令 $\mathcal{M} = \langle \mathcal{D}, 1^\mathcal{M}, \text{equiv}^\mathcal{M}, \cdot^\mathcal{M}, =^\mathcal{M} \rangle$ 为理论 $MEA$ 的任意一个模型。
*   根据公理 **EA3** 和 **EA4**，代数结构 $\mathcal{G} = \langle \mathcal{D}, \cdot^\mathcal{M} \rangle$ 构成一个群。
*   我们用 $e$ 表示该群的单位元 (identity element)。
*   我们用 $x^{-1}$ 表示元素 $x \in \mathcal{D}$ 的群逆元 (group inverse)。

---

#### **引理 4.1.**
$\forall x \forall z ( \text{equiv}(x, z) = \cdot(x, \text{equiv}(e, z)) )$

**证明:**
1.  $\forall a \forall b \forall c ( \text{equiv}(\cdot(a, b), c) = \cdot(a, \text{equiv}(b, c)) )$
    (公理 EA5)
2.  $\text{equiv}(\cdot(x, e), z) = \cdot(x, \text{equiv}(e, z))$
    (对 1. 进行全称实例化: $a/x, b/e, c/z$)
3.  $\cdot(x, e) = x$
    (群单位元 $e$ 的定义)
4.  $\text{equiv}(x, z) = \cdot(x, \text{equiv}(e, z))$
    (将 3. 代入 2.)
5.  $\forall x \forall z ( \text{equiv}(x, z) = \cdot(x, \text{equiv}(e, z)) )$
    (全称概括)

---

#### **引理 4.2.**
$\forall z ( \text{equiv}(e, z) = \cdot(z, 1) )$

**证明:**
1.  $\forall x \forall y ( \text{equiv}(x, y) = \text{equiv}(y, x) )$
    (公理 EA2)
2.  $\forall x \forall z ( \text{equiv}(x, z) = \cdot(x, \text{equiv}(e, z)) )$
    (引理 4.1)
3.  $\text{equiv}(x, z) = \text{equiv}(z, x)$
    (对 1. 进行全称实例化)
4.  $\text{equiv}(x, z) = \cdot(x, \text{equiv}(e, z))$
    (对 2. 进行全称实例化)
5.  $\text{equiv}(z, x) = \cdot(z, \text{equiv}(e, x))$
    (对 2. 进行全称实例化: $x/z, z/x$)
6.  $\cdot(x, \text{equiv}(e, z)) = \cdot(z, \text{equiv}(e, x))$
    (根据 3., 4., 5. 和等词的传递性)
7.  $\cdot(e, \text{equiv}(e, z)) = \cdot(z, \text{equiv}(e, e))$
    (对 6. 进行全称实例化: $x/e$)
8.  $\cdot(e, \text{equiv}(e, z)) = \text{equiv}(e, z)$
    (群单位元 $e$ 的定义)
9.  $\text{equiv}(e, e) = 1$
    (公理 EA1: $\text{equiv}(x, y) = 1 \leftrightarrow x = y$)
10. $\text{equiv}(e, z) = \cdot(z, 1)$
    (将 8., 9. 代入 7.)
11. $\forall z ( \text{equiv}(e, z) = \cdot(z, 1) )$
    (全称概括)

---

#### **引理 4.3.**
$\forall x \forall y ( \cdot(x, 1) = \cdot(y, 1) \to x = y )$

**证明:**
1.  假设 $\cdot(x, 1) = \cdot(y, 1)$。
    (条件证明的假设)
2.  $\text{equiv}(e, x) = \cdot(x, 1)$
    (引理 4.2)
3.  $\text{equiv}(e, y) = \cdot(y, 1)$
    (引理 4.2)
4.  $\text{equiv}(e, x) = \text{equiv}(e, y)$
    (根据 1., 2., 3. 和等词的传递性)
5.  $\text{equiv}(a, x) = \cdot(a, \text{equiv}(e, x))$
    (引理 4.1)
6.  $\text{equiv}(a, y) = \cdot(a, \text{equiv}(e, y))$
    (引理 4.1)
7.  $\cdot(a, \text{equiv}(e, x)) = \cdot(a, \text{equiv}(e, y))$
    (根据 4. 和等词公理 A7)
8.  $\text{equiv}(a, x) = \text{equiv}(a, y)$
    (根据 5., 6., 7. 和等词的传递性)
9.  $\text{equiv}(x, x) = \text{equiv}(x, y)$
    (对 8. 进行全称实例化: $a/x$)
10. $\text{equiv}(x, x) = 1$
    (公理 EA1)
11. $1 = \text{equiv}(x, y)$
    (将 10. 代入 9.)
12. $x = y$
    (公理 EA1: $1 = \text{equiv}(x, y) \leftrightarrow x = y$)
13. $\cdot(x, 1) = \cdot(y, 1) \to x = y$
    (条件证明结论)
14. $\forall x \forall y ( \cdot(x, 1) = \cdot(y, 1) \to x = y )$
    (全称概括)

#### **定理 4.4. 任何模型的代数结构均为布尔群 (Boolean Group)**
对于理论 $MEA$ 的任意模型，其代数结构 $\mathcal{G} = \langle \mathcal{D}, \cdot^\mathcal{M} \rangle$ 是一个布尔群。即，群中任意元素均为其自身的逆。
$\forall x ( \cdot(x, x) = e )$

**证明:**
1.  $\forall z (\text{equiv}(z, z) = 1)$
    (公理 EA1)
2.  $\forall x \forall z (\text{equiv}(x, z) = \cdot(x, \cdot(z, 1)))$
    (结合引理 4.1 和引理 4.2)
3.  $\text{equiv}(z, z) = \cdot(z, \cdot(z, 1))$
    (对 2. 进行全称实例化: $x/z$)
4.  $1 = \cdot(z, \cdot(z, 1))$
    (将 1. 代入 3.)
5.  $\cdot(z^{-1}, 1) = \cdot(z^{-1}, \cdot(z, \cdot(z, 1)))$
    (在 4. 两边同时左乘 $z^{-1}$)
6.  $\cdot(z^{-1}, \cdot(z, \cdot(z, 1))) = \cdot(\cdot(z^{-1}, z), \cdot(z, 1))$
    (公理 EA4 - 结合律)
7.  $\cdot(z^{-1}, z) = e$
    (群逆元的定义)
8.  $\cdot(z^{-1}, 1) = \cdot(e, \cdot(z, 1))$
    (将 6., 7. 代入 5.)
9.  $\cdot(e, \cdot(z, 1)) = \cdot(z, 1)$
    (群单位元的定义)
10. $\cdot(z^{-1}, 1) = \cdot(z, 1)$
    (根据 8., 9. 和等词的传递性)
11. $z^{-1} = z$
    (根据 10. 和引理 4.3)
12. $\cdot(z, z) = \cdot(z, z^{-1})$
    (在 11. 两边同时左乘 $z$)
13. $\cdot(z, z^{-1}) = e$
    (群逆元的定义)
14. $\cdot(z, z) = e$
    (根据 12., 13. 和等词的传递性)
15. $\forall x (\cdot(x, x) = e)$
    (全称概括，重命名变量 $z$ 为 $x$)
