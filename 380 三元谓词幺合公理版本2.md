#### **I. 语言 (Language)**

本理论的形式语言 L 基于含等词的一阶谓词逻辑。
*   **非逻辑符号:**
    1.  一个三元谓词符号：`P`
*   **逻辑符号:**
    1.  二元谓词符号：` = `
    2.  命题联结词： `¬`, `→`
    3.  量词： `∀`
    4.  变量符号： `x, y, z, ...`
    5.  辅助符号： `(`, `)`

*   **标准缩写:**
    *   `φ ∧ ψ`  缩写为  `¬(φ → ¬ψ)`
    *   `φ ∨ ψ`  缩写为  `¬φ → ψ`
    *   `φ ↔ ψ`  缩写为  `(φ → ψ) ∧ (ψ → φ)`
    *   `∃x φ`  缩写为  `¬∀x ¬φ`
    *   `∃!x φ(x)` 缩写为 `∃x (φ(x) ∧ ∀y(φ(y) → y=x))`

#### **II. 逻辑框架 (Logical Framework)**

本理论采用标准的一阶逻辑公理和推理规则，并包含等词公理：
1.  `∀x (x=x)`
2.  `∀x ∀y (x=y → (φ(x) → φ(y)))` 对于语言 L 中的任意公式 `φ`。

#### **III. 定义 (Definitions)**

**定义 D1：函数 (Function)**
`IsFunction(f) := ∀x ∃!y P(f, x, y)`

**定义 D2: 零 (Zero)**
`IsZero(z) := ∀f, x ( ∃y (P(z, f, y) ∧ P(y, x, x)) )`

**定义 D3: 后继 (Successor)**
`IsSuccessor(s_c) := ∀n, f, x, y, z, w ( P(n, f, y) ∧ P(y, x, z) → ∃s (P(s_c, n, s) ∧ ∃u (P(s, f, u) ∧ P(u, x, w) ∧ P(f, z, w))) )`

**定义 D4：归纳性质持有者 (Inductive Property Holder)**
`IsInductive(K) :=`
` (∀z (IsZero(z) → P(K, z, z))) ∧`
` (∀s_c (IsSuccessor(s_c) → ∀n (P(K, n, n) → ∃s (P(s_c, n, s) ∧ P(K, s, s)))))`

**定义 D5：索引 (Index)**
`IsIndex(i) := ∀K ( IsInductive(K) → P(K, i, i) )`

**定义 D6：索引序 (Index Order)**
`i ≤ n := ∀K ( (P(K, i, i) ∧ (∀s_c (IsSuccessor(s_c) → ∀x (P(K, x, x) → ∃s (P(s_c, x, s) ∧ P(K, s, s)))))) → P(K, n, n) )`

**定义 D7：n元关系 (n-ary Relation)**
`IsRelation(R, n) := IsIndex(n) ∧ P(α, R, n) ∧ ∀i ( (IsIndex(i) ∧ i ≤ n) → ∃!x ∃π (P(Π, i, π) ∧ P(π, R, x)) )`

**定义 D8：单射性 (Injectivity)**
`IsInjective(f) := ∀x₁, x₂ ( (∃y (P(f, x₁, y) ∧ P(f, x₂, y))) → x₁ = x₂ )`

**定义 D9：不在值域中 (Not In Range)**
`IsNotInRange(z, f) := ∀x ¬P(f, x, z)`

**定义 D10：计算同构 (Computational Isomorphism)**
`IsIsomorphism(f, g, a, b) :=`
`IsFunction(f) ∧ IsFunction(g) ∧`
`∀x ( ∃y (P(a, x, y)) → ∃z (P(f, y, z) ∧ P(b, x, z)) ) ∧`
`∀x ( ∃y (P(b, x, y)) → ∃z (P(g, y, z) ∧ P(a, x, z)) ) ∧`
`∀x ( ∃y (P(a, x, y)) → ∃z, w (P(f, y, z) ∧ P(g, z, w) ∧ y = w) ) ∧`
`∀x ( ∃y (P(b, x, y)) → ∃z, w (P(g, y, z) ∧ P(f, z, w) ∧ y = w) )`

#### **IV. 公理 (Axioms)**

**公理 A1：函数性唯一公理 (Axiom of Functional Uniqueness)**
`∀a, b, c₁, c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`

**公理 A2：投影函数构造公理 (Axiom of Projection Function Construction)**
`∃Π ( IsFunction(Π) ∧ ∀i (IsIndex(i) → ∃!π (IsFunction(π) ∧ P(Π, i, π))) )`

**公理 A3：元数函数公理 (Axiom of Arity Function)**
`∃α (IsFunction(α))`

**公理 A4：计算幺合公理 (Axiom of Computational Univalence)**
`∀a, b ( (∃f, g IsIsomorphism(f, g, a, b)) ↔ a = b )`

**公理 A5：共轭公理 (Axiom of Conjugation)**
`∀f ∃g ( IsFunction(f) → (IsFunction(g) ∧ ∀a, b ( P(f, a, b) ↔ P(a, g, b) ) ) )`

### **V. 一个退化的模型语义 (Model Semantics)**

为了证明该形式系统的可靠性、非平凡性与一致性，我们构建一个模型 **M**，尽管它不是我们预期的标准模型。

**定义 V.1：模型 M**
模型 **M** 由一个论域 `D` 和一个解释函数 `I` 组成，记为 **M** = <`D`, `I`>。
1.  **论域 (Domain of Discourse):**
    `D` 定义为正整数集合 `ℕ+`。
    `D = {1, 2, 3, ...}`

2.  **解释 (Interpretation):**
    解释函数 `I` 将语言 `L` 中的非逻辑符号映射到论域 `D` 上的对象、函数或关系。
    *   对于语言 `L` 中的常量符号（本系统无），`I(c) ∈ D`。
    *   对于n元函数符号（本系统无），`I(f)` 是一个从 `Dⁿ` 到 `D` 的函数。
    *   对于n元谓词符号，`I(R)` 是 `Dⁿ` 的一个子集。

    在本系统中，对三元谓词符号 `P` 的解释 `P^M` 如下：
    `P^M = { (a, b, c) | a, b, c ∈ D ∧ a * b = c }`
    其中 `*` 是 `ℕ+` 上的标准乘法运算。

因此，对于任意 `a, b, c ∈ D`，**M** 满足 `P(a, b, c)`，记为 `M ⊨ P(a, b, c)`，当且仅当 `a * b = c`。

### **VI. 可靠性、非平凡性与一致性证明**

本节将证明在上一节定义的模型 **M** 中，所有公理均为真，且该模型是非平凡的。由此可得出该形式系统是一致的。

#### **VI.1 可靠性证明 (Proof of Soundness)**

我们逐一验证公理 A1 至 A5 在模型 **M** 中为真。

**引理 VI.1.1：`IsFunction` 在 M 中的解释**
对于任意 `f ∈ D`，`M ⊨ IsFunction(f)`。
*   **证明:**
    1.  `IsFunction(f)` 的定义为 `∀x ∃!y P(f, x, y)`。
    2.  在模型 **M** 中，此定义被解释为 `∀x ∈ D, ∃!y ∈ D (f * x = y)`。
    3.  根据 `ℕ+` 的代数性质，对于任意给定的 `f, x ∈ ℕ+`，它们的乘积 `f * x` 是一个唯一的、确定的正整数。
    4.  因此，该 `y` 存在且唯一。故 `M ⊨ IsFunction(f)` 对所有 `f ∈ D` 成立。

**引理 VI.1.2：`IsZero`, `IsSuccessor`, `IsIndex` 在 M 中的解释**
在模型 **M** 中，满足 `IsZero`, `IsSuccessor`, `IsIndex` 的元素集合为空。
*   **证明 `IsZero`:**
    1.  `IsZero(z)` 定义为 `∀f, x ( ∃y (P(z, f, y) ∧ P(y, x, x)) )`。
    2.  在 **M** 中，这被解释为 `∀f, x ∈ D ( ∃y ∈ D (z * f = y ∧ y * x = x) )`。
    3.  合并条件得 `∀f, x ∈ D ( (z * f) * x = x )`。
    4.  由于此式对所有 `x ∈ D` 成立，必须有 `z * f = 1` 对所有 `f ∈ D` 成立。
    5.  取 `f=2`，则要求 `z * 2 = 1`，解得 `z = 1/2`。
    6.  `1/2 ∉ D`。因此，不存在 `z ∈ D` 满足 `IsZero(z)`。
*   **证明 `IsSuccessor`:**
    1.  我们的目标是证明对于任意 `s_c ∈ D`，`M ⊨ IsSuccessor(s_c)` 为假。即，不存在 `s_c ∈ D` 满足该定义。
    2.  根据定义 D3，`IsSuccessor(s_c)` 为真，当且仅当以下公式在模型 **M** 中为真：
        `∀n, f, x, y, z, w ( P(n, f, y) ∧ P(y, x, z) → ∃s (P(s_c, n, s) ∧ ∃u (P(s, f, u) ∧ P(u, x, w) ∧ P(f, z, w))) )`
    3.  在模型 **M** 中，该公式被解释为，对于任意 `s_c ∈ D`，以下关于 `ℕ+` 的陈述必须成立：
        `∀n, f, x, y, z, w ∈ ℕ+ ( (n*f = y ∧ y*x = z) → ∃s ∈ ℕ+ (s_c*n = s ∧ ∃u ∈ ℕ+ (s*f = u ∧ u*x = w ∧ f*z = w)) )`
    4.  我们分析此蕴含式。对于任意给定的 `n, f, x ∈ ℕ+`，蕴含式的前件 `n*f = y ∧ y*x = z` 总是可以被满足，因为 `ℕ+` 在乘法下是封闭的。我们可以通过代入 `y = n*f` 来消除变量 `y`，得到 `(n*f)*x = z`。
    5.  现在分析蕴含式的后件。后件断言存在 `s, u, w ∈ ℕ+` 满足一系列等式。我们可以通过代入来消除存在量词：
        a.  将 `s_c*n = s` 代入 `s*f = u`，得到 `(s_c*n)*f = u`。
        b.  将 `u` 的表达式代入 `u*x = w`，得到 `((s_c*n)*f)*x = w`。
        c.  因此，后件等价于存在一个 `w`，使得 `w = ((s_c*n)*f)*x` 并且 `w = f*z`。这蕴含了 `((s_c*n)*f)*x = f*z`。
    6.  将前件中 `z = (n*f)*x` 的表达式代入到步骤 5 推导出的等式中：
        `((s_c*n)*f)*x = f*((n*f)*x)`
    7.  此等式必须对所有 `n, f, x ∈ ℕ+` 成立。根据 `ℕ+` 上乘法的结合律，此等式化为：
        `s_c * n * f * x = f * n * f * x`
    8.  由于此等式对所有 `n, f, x ∈ ℕ+` 均成立，我们可以应用 `ℕ+` 中的乘法消去律，消去等式两边的公因子 `n*f*x`（该因子恒为正整数，不为零）：
        `s_c = f`
    9.  此结果 `s_c = f` 是从 `∀n, f, x` 的前提下推导出来的。这意味着，一个固定的 `s_c` 必须等于任意选择的 `f ∈ ℕ+`。
    10. 这是一个矛盾。例如，我们可取 `f=1` 和 `f=2`。`s_c = 1` 和 `s_c = 2` 不可能同时成立。
    11. 因此，不存在任何一个 `s_c ∈ D` 能使得步骤 3 中的全称量化陈述为真。故满足 `IsSuccessor(s_c)` 的元素集合为空。
*   **证明 `IsIndex`:**
    1.  `IsInductive(K)` 定义为 `(∀z (IsZero(z) → ...)) ∧ (∀s_c (IsSuccessor(s_c) → ...))`。
    2.  由于不存在满足 `IsZero(z)` 和 `IsSuccessor(s_c)` 的元素，这两个蕴含式的前件恒为假，故蕴含式空洞地为真。
    3.  因此，`M ⊨ IsInductive(K)` 对所有 `K ∈ D` 成立。
    4.  `IsIndex(i)` 定义为 `∀K ( IsInductive(K) → P(K, i, i) )`。
    5.  在 **M** 中，这简化为 `∀K ∈ D ( True → (K * i = i) )`，即 `∀K ∈ D (K * i = i)`。
    6.  取 `K=2`，则 `2 * i = i`，这要求 `i=0`。
    7.  `0 ∉ D`。因此，不存在 `i ∈ D` 满足 `IsIndex(i)`。

**定理 VI.1.3：公理 A1-A5 在 M 中为真**

*   **证明 A1 (函数性唯一公理):** `M ⊨ ∀a, b, c₁, c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`
    1.  在 **M** 中，该公理被解释为 `∀a, b, c₁, c₂ ∈ D ( (a * b = c₁ ∧ a * b = c₂) → c₁ = c₂ )`。
    2.  这是 `ℕ+` 上等词的传递性律和乘法作为函数的良定义性的直接结果。该陈述为真。

*   **证明 A2 (投影函数构造公理):** `M ⊨ ∃Π ( IsFunction(Π) ∧ ∀i (IsIndex(i) → ∃!π (IsFunction(π) ∧ P(Π, i, π))) )`
    1.  根据引理 VI.1.1，`IsFunction(Π)` 对任意 `Π ∈ D` 为真。
    2.  根据引理 VI.1.2，`IsIndex(i)` 对任意 `i ∈ D` 为假。
    3.  因此，蕴含式 `IsIndex(i) → ...` 的前件恒为假，故整个蕴含式对任意 `i` 空洞地为真。
    4.  公理简化为 `∃Π ∈ D ( True ∧ True )`，即 `∃Π ∈ D`。
    5.  由于论域 `D = ℕ+` 非空，这样的 `Π` 存在（例如，取 `Π=1`）。该陈述为真。

*   **证明 A3 (元数函数公理):** `M ⊨ ∃α (IsFunction(α))`
    1.  根据引理 VI.1.1，`M ⊨ IsFunction(α)` 对所有 `α ∈ D` 成立。
    2.  由于 `D` 非空，至少存在一个 `α`。该陈述为真。

*   **证明 A4 (计算幺合公理):** `M ⊨ ∀a, b ( (∃f, g IsIsomorphism(f, g, a, b)) ↔ a = b )`
    1.  首先分析 `IsIsomorphism(f, g, a, b)` 在 **M** 中的条件：
        a. `IsFunction(f)` 和 `IsFunction(g)` 恒为真。
        b. `∀x ( ∃y (a*x=y) → ∃z (f*y=z ∧ b*x=z) )` 化简为 `f*(a*x) = b*x`，即 `(f*a)*x = b*x`，得出 `f*a=b`。
        c. `∀x ( ∃y (b*x=y) → ∃z (g*y=z ∧ a*x=z) )` 化简为 `g*(b*x) = a*x`，即 `(g*b)*x = a*x`，得出 `g*b=a`。
        d. 逆函数条件 `g*(f*y)=y` 对于 `y` 在 `a` 的倍数集合中成立，要求 `g*f=1`。
    2.  因此，`M ⊨ ∃f, g IsIsomorphism(f, g, a, b)` 等价于 `∃f, g ∈ D ( f*a=b ∧ g*b=a ∧ f*g=1 )`。
    3.  在 `D = ℕ+` 中，满足 `f*g=1` 的唯一解是 `f=1` 且 `g=1`。
    4.  将 `f=1, g=1` 代入，条件变为 `1*a=b` 和 `1*b=a`，即 `a=b`。
    5.  所以，`∃f, g IsIsomorphism(f, g, a, b)` 为真当且仅当 `a=b`。
    6.  这与公理 A4 的 `↔` 陈述完全一致。该陈述为真。

*   **证明 A5 (共轭公理):** `M ⊨ ∀f ∃g ( IsFunction(f) → (IsFunction(g) ∧ ∀a, b ( P(f, a, b) ↔ P(a, g, b) ) ) )`
    1.  `IsFunction` 条件恒为真，公理简化为 `∀f ∈ D ∃g ∈ D ∀a, b ∈ D ( (f*a=b) ↔ (a*g=b) )`。
    2.  对任意给定的 `f ∈ D`，我们选择 `g = f`。由于 `f ∈ D`，所以 `g ∈ D`。
    3.  待证陈述变为 `∀f ∈ D ∀a, b ∈ D ( (f*a=b) ↔ (a*f=b) )`。
    4.  根据 `ℕ+` 上乘法的交换律，`f*a = a*f` 恒成立。
    5.  因此，` (f*a=b) ↔ (a*f=b) ` 是一个重言式。该陈述为真。

**结论：** 模型 **M** 满足该形式系统的所有公理。

#### **VI.2 非平凡性证明 (Proof of Non-triviality)**

1.  一个模型是平凡的，当且仅当其论域 `D` 只包含一个元素。
2.  模型 **M** 的论域为 `D = ℕ+ = {1, 2, 3, ...}`。
3.  `D` 包含多于一个元素（例如 `1 ∈ D` 且 `2 ∈ D` 且 `1 ≠ 2`）。
4.  因此，模型 **M** 是非平凡的。

#### **VI.3 一致性证明 (Proof of Consistency)**

1.  根据哥德尔完全性定理，一个一阶理论是一致的（syntactically consistent），当且仅当它有一个模型（semantically consistent）。
2.  在 VI.1 中，我们证明了 **M** 是该形式系统的一个模型（`M ⊨ T`，其中 `T` 是公理集合）。
3.  在 VI.2 中，我们证明了该模型是**非平凡**的。
4.  因为该形式系统存在一个模型，所以它不可能推导出任何形如 `φ ∧ ¬φ` 的矛盾。
5.  因此，该形式系统是一致的。
