
#### **1. 语言 (Language)**

本系统的形式语言 **L** 是一种含等词的一阶谓词逻辑语言。

1.  **非逻辑符号:**
    *   一个三元谓词符号：`P`

2.  **逻辑符号:**
    *   二元谓词符号：` = `
    *   命题联结词： `¬`, `→`
    *   量词： `∀`
    *   变量符号： `v₀, v₁, v₂, ...`
    *   辅助符号： `(`, `)`

#### **2. 构造规则 (Formation Rules)**

1.  **项 (Terms):** 本语言中的项仅包含变量 `v₀, v₁, v₂, ...`。
2.  **原子公式 (Atomic Formulas):** 若 `t₁, t₂, t₃` 为项，则 `t₁ = t₂` 与 `P(t₁, t₂, t₃)` 是原子公式。
3.  **合式公式 (Well-Formed Formulas - WFFs):**
    *   所有原子公式都是合式公式。
    *   若 `φ` 和 `ψ` 是合式公式，则 `(¬φ)` 和 `(φ → ψ)` 是合式公式。
    *   若 `φ` 是合式公式且 `x` 是一个变量，则 `(∀x φ)` 是合式公式。

#### **3. 定义 (Definitions)**

以下为标准逻辑符号的缩写：
*   `φ ∧ ψ`   :=   `¬(φ → ¬ψ)`
*   `φ ∨ ψ`   :=   `(¬φ) → ψ`
*   `φ ↔ ψ`   :=   `(φ → ψ) ∧ (ψ → φ)`
*   `∃x φ`    :=   `¬∀x (¬φ)`
*   `∃!x φ`   :=   `∃x (φ ∧ ∀y (φ[y/x] → y=x))`

以下为本系统的特有定义：

**D1: IsFunction(f)**
`∀x ∃!y P(f, x, y)`

#### **4. 公理系统 (Axiom System)**

本系统包含标准的一阶逻辑公理模式与等词公理。

1.  **逻辑公理:**
    *   所有重言式实例。
    *   `∀x φ(x) → φ(t)` (当项 `t` 在 `φ(x)` 中对 `x` 可代入时)。
    *   `∀x (φ → ψ) → (∀x φ → ∀x ψ)`
    *   `φ → ∀x φ` (当 `x` 在 `φ` 中不是自由变量时)。

2.  **等词公理:**
    *   `∀x (x=x)`
    *   `∀x ∀y (x=y → (φ(x) → φ(y)))`

3.  **特有公理:**

    **A1: Uniqueness**
    `∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`

    **A2: Extensionality**
    `∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`

	**A3: 对称应用公理 (Axiom of Symmetric Application)**
	`∃p ( IsFunction(p) ∧ ∀f ( IsFunction(f) → (∃y P(f, p, y) ↔ ∃y P(p, f, y)) ∧ (∀y₁ ∀y₂ (P(f, p, y₁) ∧ P(p, f, y₂)) → y₁ = y₂) ) )`

	**A4: 层级否定公理 (Axiom of Hierarchical Negation)**
	`∃n ( IsFunction(n) ∧ ∀x (∃!y P(n, x, y)) ∧ ∀x ∀y (P(n, x, y) → x ≠ y) ∧ ∀x ∀y ∀z ((P(n, x, y) ∧ P(n, y, z)) → x ≠ z) )`

#### **5. 推理规则 (Rules of Inference)**

1.  **分离规则 (Modus Ponens):**
    从 `φ` 和 `φ → ψ`，可推出 `ψ`。

2.  **概括规则 (Generalization):**
    从 `φ`，可推出 `∀x φ`。

#### **6. 模型 (Model)**

本节为前述形式系统提供一个非平凡模型 **M**。

##### **6.1 模型 M 的定义**

该模型 **M** 是一个结构 `⟨D, I⟩`，其中 `D` 为论域 (Domain)，`I` 为释义函数 (Interpretation function)。

1.  **论域 (Domain):**
    `D = {0, 1, 2}`

2.  **释义 (Interpretation):**
    释义函数 `I` 将语言 **L** 中的非逻辑符号映射到论域 `D` 上的对象和关系。
    *   **谓词符号 `P` 的释义:**
        `I(P)` (记作 `Pᴹ`) 是 `D × D × D` 的一个子集。`Pᴹ(a, b, c)` 为真，当且仅当 `(a, b, c)` 属于以下集合：
        `Pᴹ = {`
        `(0, 0, 1), (0, 1, 2), (0, 2, 0),`
        `(1, 0, 2), (1, 1, 1), (1, 2, 0),`
        `(2, 0, 0), (2, 1, 0), (2, 2, 2)`
        `}`

##### **6.2 模型 M 的验证**

我们将证明模型 **M** = `⟨{0, 1, 2}, Pᴹ}⟩` 满足所有特有公理。首先验证定义 **D1** 的性质。

**D1: IsFunction(f)**
公式 `IsFunction(f)` 即 `∀x ∃!y P(f, x, y)`。
在模型 **M** 中，对 `D` 中的每一个元素 `f` 和 `x`，都存在唯一的 `y` 使得 `(f, x, y) ∈ Pᴹ`。
*   对于 `f=0`：`(0,0,1)`, `(0,1,2)`, `(0,2,0)`。对每个 `x` 都有唯一 `y`。
*   对于 `f=1`：`(1,0,2)`, `(1,1,1)`, `(1,2,0)`。对每个 `x` 都有唯一 `y`。
*   对于 `f=2`：`(2,0,0)`, `(2,1,0)`, `(2,2,2)`。对每个 `x` 都有唯一 `y`。
因此，`M ⊨ IsFunction(0)`, `M ⊨ IsFunction(1)` 且 `M ⊨ IsFunction(2)`。

**A1: Uniqueness**
`∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`
该公理要求对于任意给定的 `a` 和 `b`，最多只有一个 `c` 使得 `(a, b, c) ∈ Pᴹ`。通过检视 `Pᴹ` 的定义，我们可以看到对于 `D × D` 中的每一个序对 `(a, b)`，都恰好只有一个 `c` 使得 `(a, b, c) ∈ Pᴹ`。因此，该公理成立。

**A2: Extensionality**
`∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`
`→` 方向是等词的标准性质。我们验证 `←` 方向，即证明对于 `D` 中任意两个不同元素 `a, b`，括号内的合取式为假。

*   **比较 a=0 和 b=1:**
    令 `x=1`。`∃y P(0, x, y)` 为真 (因为 `(0, 1, 2) ∈ Pᴹ`)，而 `∃y P(1, x, y)` 也为真 (因为 `(1, 1, 1) ∈ Pᴹ`)。
    然而，`P(0, 1, 1)` 为假，`P(1, 1, 1)` 为真。因此 `P(0, 1, 1) ↔ P(1, 1, 1)` 为假。
    所以 `∀x ∀y (P(0, x, y) ↔ P(1, x, y))` 为假，合取式为假。

*   **比较 a=0 和 b=2:**
    令 `x=0`。`P(0, 0, 1)` 为真，`P(2, 0, 1)` 为假。因此 `P(0, 0, 1) ↔ P(2, 0, 1)` 为假。
    所以 `∀x ∀y (P(0, x, y) ↔ P(2, x, y))` 为假，合取式为假。

*   **比较 a=1 和 b=2:**
    令 `x=1`。`P(1, 1, 1)` 为真，`P(2, 1, 1)` 为假。因此 `P(1, 1, 1) ↔ P(2, 1, 1)` 为假。
    所以 `∀x ∀y (P(1, x, y) ↔ P(2, x, y))` 为假，合取式为假。

因此，公理 A2 在模型 **M** 中成立。

**A3: 对称应用公理 (Axiom of Symmetric Application)**
`∃p ( IsFunction(p) ∧ ∀f ( IsFunction(f) → (∃y P(f, p, y) ↔ ∃y P(p, f, y)) ∧ (∀y₁ ∀y₂ (P(f, p, y₁) ∧ P(p, f, y₂)) → y₁ = y₂) ) )`
我们断言 `p = 1` 是该公理的一个满足实例。
1.  `IsFunction(1)` 已被证明为真。
2.  我们需要对 `f ∈ {0, 1, 2}` 验证 `(...)` 部分。

*   **f = 0:**
    *   `∃y P(0, 1, y)` 为真，因为 `(0, 1, 2) ∈ Pᴹ`。
    *   `∃y P(1, 0, y)` 为真，因为 `(1, 0, 2) ∈ Pᴹ`。
    *   `P(0, 1, y₁)` 蕴含 `y₁=2`。 `P(1, 0, y₂)` 蕴含 `y₂=2`。
    *   结论 `y₁ = y₂` (即 `2=2`) 为真。

*   **f = 1:**
    *   `∃y P(1, 1, y)` 为真，因为 `(1, 1, 1) ∈ Pᴹ`。
    *   `∃y P(1, 1, y)` 为真，因为 `(1, 1, 1) ∈ Pᴹ`。
    *   `P(1, 1, y₁)` 蕴含 `y₁=1`。 `P(1, 1, y₂)` 蕴含 `y₂=1`。
    *   结论 `y₁ = y₂` (即 `1=1`) 为真。

*   **f = 2:**
    *   `∃y P(2, 1, y)` 为真，因为 `(2, 1, 0) ∈ Pᴹ`。
    *   `∃y P(1, 2, y)` 为真，因为 `(1, 2, 0) ∈ Pᴹ`。
    *   `P(2, 1, y₁)` 蕴含 `y₁=0`。 `P(1, 2, y₂)` 蕴含 `y₂=0`。
    *   结论 `y₁ = y₂` (即 `0=0`) 为真。

由于存在 `p=1` 满足条件，公理 A3 在模型 **M** 中成立。

**A4: 层级否定公理 (Axiom of Hierarchical Negation)**
`∃n ( IsFunction(n) ∧ ∀x ∀y (P(n, x, y) → x ≠ y) ∧ ∀x ∀y ∀z ((P(n, x, y) ∧ P(n, y, z)) → x ≠ z) )`
我们断言 `n = 0` 是该公理的一个满足实例。
1.  `IsFunction(0)` 已被证明为真。
2.  验证 `∀x ∀y (P(0, x, y) → x ≠ y)` (无不动点)。
    *   `x=0`: `P(0,0,1)` 为真，结论 `0 ≠ 1` 为真。
    *   `x=1`: `P(0,1,2)` 为真，结论 `1 ≠ 2` 为真。
    *   `x=2`: `P(0,2,0)` 为真，结论 `2 ≠ 0` 为真。
    该子句成立。
3.  验证 `∀x ∀y ∀z ((P(0, x, y) ∧ P(0, y, z)) → x ≠ z)` (无二阶不动点)。
    *   `x=0`: `P(0,0,y)` 蕴含 `y=1`。`P(0,1,z)` 蕴含 `z=2`。结论 `0 ≠ 2` 为真。
    *   `x=1`: `P(0,1,y)` 蕴含 `y=2`。`P(0,2,z)` 蕴含 `z=0`。结论 `1 ≠ 0` 为真。
    *   `x=2`: `P(0,2,y)` 蕴含 `y=0`。`P(0,0,z)` 蕴含 `z=1`。结论 `2 ≠ 1` 为真。
    该子句成立。

由于存在 `n=0` 满足条件，公理 A4 在模型 **M** 中成立。

综上所述，模型 **M** = `⟨{0, 1, 2}, Pᴹ}⟩` 是该形式系统的一个非平凡模型。

#### **7. 无限模型与无穷公理 (Infinite Model and Axiom of Infinity)**

本节为前述形式系统构建一个非平凡的无限模型 **$M_∞$**，并给出一个在该模型中成立而在任何有限模型中均不成立的公理。

##### **7.1 无限模型 $M_∞$ 的定义**

该模型 **$M_∞$** 是一个结构 `⟨D, I⟩`。

1.  **论域 (Domain):**
    `D = ℕ = {0, 1, 2, ...}`

2.  **释义 (Interpretation):**
    谓词符号 `P` 的释义 `I(P)` (记作 `Pᴹ`) 是 `D × D × D` 的一个子集，定义如下：
    `Pᴹ = { (a, b, c) ∈ ℕ³ | a + b = c }`

##### **7.2 模型 $M_∞$ 的验证**

我们将证明模型 **$M_∞$** = `⟨ℕ, Pᴹ⟩` 满足所有特有公理 A1, A2, A3, A4。

**预备引理： `M_∞ ⊨ ∀f IsFunction(f)`**
*   证明：`IsFunction(f)` 为 `∀x ∃!y P(f, x, y)`。
*   在 `M_∞` 中，此式释义为 `∀x ∈ ℕ, ∃!y ∈ ℕ (f + x = y)`。
*   **存在性**：对任意 `f, x ∈ ℕ`，根据自然数加法的封闭性，`f + x` 的和存在且为自然数。令 `y = f + x`。
*   **唯一性**：若 `f + x = y₁` 且 `f + x = y₂`，则根据等词的传递性，`y₁ = y₂`。
*   故对 `D` 中任意元素 `f`，`IsFunction(f)` 均成立。

**A1: Uniqueness**
`∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`
*   设 `a, b, c₁, c₂ ∈ ℕ`。
*   假设 `M_∞ ⊨ P(a, b, c₁) ∧ P(a, b, c₂)`。
*   根据 `Pᴹ` 的定义，`a + b = c₁` 且 `a + b = c₂`。
*   因此 `c₁ = c₂`。
*   故 `M_∞ ⊨ A1`。

**A2: Extensionality**
`∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`
*   `→` 方向：由等词公理直接可得。
*   `←` 方向：假设 `a ≠ b` 且 `a, b ∈ ℕ`。需证 `(...)` 为假。
    *   考察 `∀x ∀y (P(a, x, y) ↔ P(b, x, y))`。
    *   此式在 `M_∞` 中释义为 `∀x ∀y ( (a + x = y) ↔ (b + x = y) )`。
    *   令 `x = 0`，则 `∀y ( (a = y) ↔ (b = y) )`。
    *   令 `y = a`，则 `(a = a) ↔ (b = a)`，即 `⊤ ↔ ⊥`，此为假。
    *   因此 `∀x ∀y (P(a, x, y) ↔ P(b, x, y))` 为假，从而整个合取式为假。
*   故 `M_∞ ⊨ A2`。

**A3: 对称应用公理 (Axiom of Symmetric Application)**
`∃p ( IsFunction(p) ∧ ∀f ( IsFunction(f) → (∃y P(f, p, y) ↔ ∃y P(p, f, y)) ∧ (∀y₁ ∀y₂ (P(f, p, y₁) ∧ P(p, f, y₂)) → y₁ = y₂) ) )`
*   令 `p = 0` 作为存在量词的见证。
*   根据预备引理，`M_∞ ⊨ IsFunction(0)`。
*   设 `f ∈ ℕ` 为任意元素。根据预备引理，`M_∞ ⊨ IsFunction(f)`。故只需验证蕴含式的后件。
    1.  `M_∞ ⊨ ∃y P(f, 0, y) ↔ ∃y P(0, f, y)`
        *   `∃y P(f, 0, y)` 释义为 `∃y (f + 0 = y)`。取 `y = f`，此式为真。
        *   `∃y P(0, f, y)` 释义为 `∃y (0 + f = y)`。取 `y = f`，此式为真。
        *   `⊤ ↔ ⊤` 为真。
    2.  `M_∞ ⊨ ∀y₁ ∀y₂ ((P(f, 0, y₁) ∧ P(0, f, y₂)) → y₁ = y₂)`
        *   设 `y₁, y₂ ∈ ℕ`。假设 `P(f, 0, y₁)` 且 `P(0, f, y₂)`。
        *   则 `f + 0 = y₁` 且 `0 + f = y₂`。
        *   可得 `y₁ = f` 且 `y₂ = f`。故 `y₁ = y₂`。
*   故 `M_∞ ⊨ A3`。

**A4: 层级否定公理 (Axiom of Hierarchical Negation)**
`∃n ( IsFunction(n) ∧ ∀x ∀y (P(n, x, y) → x ≠ y) ∧ ∀x ∀y ∀z ((P(n, x, y) ∧ P(n, y, z)) → x ≠ z) )`
*   令 `n = 1` 作为存在量词的见证。
*   根据预备引理，`M_∞ ⊨ IsFunction(1)`。
    1.  `M_∞ ⊨ ∀x ∀y (P(1, x, y) → x ≠ y)`
        *   设 `x, y ∈ ℕ`。假设 `P(1, x, y)`。
        *   则 `1 + x = y`。因为 `1 > 0`，所以 `1 + x > x`，故 `y ≠ x`。
    2.  `M_∞ ⊨ ∀x ∀y ∀z ((P(1, x, y) ∧ P(1, y, z)) → x ≠ z)`
        *   设 `x, y, z ∈ ℕ`。假设 `P(1, x, y)` 且 `P(1, y, z)`。
        *   则 `1 + x = y` 且 `1 + y = z`。
        *   代入得 `1 + (1 + x) = z`，即 `2 + x = z`。
        *   因为 `2 > 0`，所以 `2 + x > x`，故 `z ≠ x`。
*   故 `M_∞ ⊨ A4`。

##### **7.3 无穷公理 (Axiom of Infinity)**

为区分有限与无限模型，我们引入以下定义与公理。

**D2: IsInjective(f)**
`∀x₁ ∀x₂ ∀y ( (P(f, x₁, y) ∧ P(f, x₂, y)) → x₁ = x₂ )`

**$A_{inf}$: 无穷公理 (Axiom of Infinity)**
`∃f ( IsFunction(f) ∧ IsInjective(f) ∧ ∃y ∀x ¬P(f, x, y) )`

**命题 7.3.1： $M_∞ ⊨ A_{inf}$**
*   证明：令 `f = 1` 作为存在量词的见证。
    1.  $M_∞ ⊨ IsFunction(1)$：已由预备引理证明。
    2.  $M_∞ ⊨ IsInjective(1)$：
        *   需证 `∀x₁ ∀x₂ ∀y ( (P(1, x₁, y) ∧ P(1, x₂, y)) → x₁ = x₂ )`。
        *   设 `x₁, x₂, y ∈ ℕ`。假设 `P(1, x₁, y)` 且 `P(1, x₂, y)`。
        *   则 `1 + x₁ = y` 且 `1 + x₂ = y`。
        *   可得 `1 + x₁ = 1 + x₂`。根据自然数加法的消去律，`x₁ = x₂`。
    3.  $M_∞ ⊨ ∃y ∀x ¬P(1, x, y)$：
        *   令 `y = 0` 作为存在量词的见证。
        *   需证 `∀x ¬P(1, x, 0)`。
        *   设 `x ∈ ℕ`。`P(1, x, 0)` 意为 `1 + x = 0`。
        *   在自然数集 `ℕ` 中，此方程无解。
        *   故 `¬P(1, x, 0)` 对所有 `x ∈ ℕ` 成立。
*   由于 `f=1` 满足全部三个条件，故 $M_∞ ⊨ A_{inf}$。

**命题 7.3.2：若 $M_{fin}$ 是任意非平凡有限模型，则 $M_{fin} ⊭ A_{inf}$**
*   证明：令 `M_fin = ⟨D_fin, Pᴹ⟩` 是满足公理 A1-A4 的模型，其中 `D_fin` 是一个基数为 `k` (`0 < k < ∞`) 的有限集合。
*   为证伪，假设 `M_fin ⊨ A_inf`。
*   则存在一个元素 `f₀ ∈ D_fin`，使得以下三条同时在 `M_fin` 中为真：
    1.  `IsFunction(f₀)`
    2.  `IsInjective(f₀)`
    3.  `∃y ∀x ¬P(f₀, x, y)`
*   由 (1)，`f₀` 在 `D_fin` 上定义了一个函数 `g: D_fin → D_fin`，其中对任意 `x ∈ D_fin`，`g(x)` 是唯一的 `y` 使得 `Pᴹ(f₀, x, y)` 为真。
*   由 (2)，函数 `g` 是单射的。因为 `IsInjective(f₀)` 保证了若 `g(x₁) = g(x₂)`，则 `x₁ = x₂`。
*   根据集合论，任何从一个有限集到其自身的单射函数必为满射函数。
*   因此，函数 `g` 必须是满射的。这意味着对于任意 `y ∈ D_fin`，存在一个 `x ∈ D_fin` 使得 `g(x) = y`。换言之，`M_fin ⊨ ∀y ∃x P(f₀, x, y)`。
*   然而，由 (3)，存在一个 `y₀ ∈ D_fin`，使得 `∀x ∈ D_fin`，`¬Pᴹ(f₀, x, y₀)`。这意味着函数 `g` 不是满射的，因为 `y₀` 不在 `g` 的值域中。
*   此为矛盾。因此，初始假设 `M_fin ⊨ A_inf` 必为假。
*   故，公理 `A_inf` 在任何非平凡的有限模型中均不成立。
