# 跨层同态演算形式系统 (System $\mathcal{H}_{CS}$)

### 第 I 部分：语法构造空间 (Syntax Space $\mathbb{T}$)

基于 ZFC 集合论，语法对象的编码空间严格落于遗传有穷集合 $V_\omega$ 中。

**定义 1.1（字母表算子）**
定义算子 $\Phi : \mathcal{P}(V_\omega) \to \mathcal{P}(V_\omega)$。对于任意 $X \subseteq V_\omega$，$\Phi(X)$ 包含且仅包含以下元素：
1. **变量**：$\forall k \in \omega, \langle 0, k \rangle \in \Phi(X)$，记作 $\mathbf{v}_k$。
2. **底元素**：$\langle 1, \emptyset \rangle \in \Phi(X)$，记作 $\mathbf{bot}$。
3. **蕴含结构**：$\forall A, B \in X, \langle 2, A, B \rangle \in \Phi(X)$，记作 $\mathbf{arr}(A, B)$。
4. **同态绑定器**：$\forall A, B \in X, \langle 3, A, B \rangle \in \Phi(X)$，记作 $\mathbf{hom}(A, B)$。
5. **降维投射**：$\forall H, A \in X, \langle 4, H, A \rangle \in \Phi(X)$，记作 $\mathbf{down}(H, A)$。

**定义 1.2（合式项集合）**
由于 $\Phi$ 是单调算子，语法空间 $\mathbb{T}$ 定义为 $V_\omega$ 中满足封闭性的最小不动点：
$$ \mathbb{T} := \bigcap \{ Y \subseteq V_\omega \mid \Phi(Y) \subseteq Y \} $$

### 第 II 部分：结构代数 (Structural Algebra)

**定义 2.1（提升算子 $\uparrow_d^c : \mathbb{T} \to \mathbb{T}$）**
对于给定的阈值 $c \in \omega$ 和提升量 $d \in \omega$，对 $A \in \mathbb{T}$ 给出结构归约定义：
1. $\uparrow_d^c(\mathbf{v}_k) = \begin{cases} \mathbf{v}_k, & \text{if } k < c \\ \mathbf{v}_{k+d}, & \text{if } k \ge c \end{cases}$
2. $\uparrow_d^c(\mathbf{bot}) = \mathbf{bot}$
3. $\uparrow_d^c(\mathbf{arr}(A, B)) = \mathbf{arr}(\uparrow_d^c(A), \uparrow_d^c(B))$
4. $\uparrow_d^c(\mathbf{hom}(A, B)) = \mathbf{hom}(\uparrow_d^{c+1}(A), \uparrow_d^{c+1}(B))$
5. $\uparrow_d^c(\mathbf{down}(H, A)) = \mathbf{down}(\uparrow_d^c(H), \uparrow_d^c(A))$

**定义 2.2（代换算子 $\cdot[\cdot]_c : \mathbb{T} \times \mathbb{T} \to \mathbb{T}$）**
对于给定的目标项 $S \in \mathbb{T}$ 和深度 $c \in \omega$，对 $A \in \mathbb{T}$ 给出结构归约定义（记为 $A[S]_c$）：
1. $\mathbf{v}_k[S]_c = \begin{cases} \mathbf{v}_k, & \text{if } k < c \\ S, & \text{if } k = c \\ \mathbf{v}_{k-1}, & \text{if } k > c \end{cases}$
2. $\mathbf{bot}[S]_c = \mathbf{bot}$
3. $\mathbf{arr}(A, B)[S]_c = \mathbf{arr}(A[S]_c, B[S]_c)$
4. $\mathbf{hom}(A, B)[S]_c = \mathbf{hom}(A[\uparrow_1^0(S)]_{c+1}, B[\uparrow_1^0(S)]_{c+1})$
5. $\mathbf{down}(H, A)[S]_c = \mathbf{down}(H[S]_c, A[S]_c)$

### 第 III 部分：层级良构语义 (Stratified Well-formedness)

**定义 3.1（层级上下文）**
上下文 $\Gamma$ 是一段表示变量层级的有穷自然数序列，$\Gamma \in \omega^{<\omega}$。记 $\Gamma(k)$ 为序列中索引为 $k$ 的元素。
记号 $x \cdot \Gamma$ 表示在上下文头部插入层级 $x$，所有原有变量的索引后移一位。

**定义 3.2（类型判断规则 $\Gamma \Vdash A : n$）**
系统通过以下规则归纳定义项 $A$ 在上下文 $\Gamma$ 下属于层级 $n \in \omega$：
1. **(WF-Var)**: $\frac{\Gamma(k) = n}{\Gamma \Vdash \mathbf{v}_k : n}$
2. **(WF-Bot)**: $\frac{}{\Gamma \Vdash \mathbf{bot} : n}$
3. **(WF-Arr)**: $\frac{\Gamma \Vdash A:n \quad \Gamma \Vdash B:n}{\Gamma \Vdash \mathbf{arr}(A, B) : n}$
4. **(WF-Hom)**: $\frac{(n+1) \cdot \Gamma \Vdash A:n \quad (n+1) \cdot \Gamma \Vdash B:n}{\Gamma \Vdash \mathbf{hom}(A, B) : n+1}$
5. **(WF-Down)**: $\frac{\Gamma \Vdash H:n+1 \quad \Gamma \Vdash A:n}{\Gamma \Vdash \mathbf{down}(H, A) : n}$

### 第 IV 部分：跨层同态演绎演算 (Deductive Calculus)

设 $\Sigma \subseteq \mathbb{T}$ 为前提集合。只有当推断目标满足 $\Gamma \Vdash A : n$，且对于 $\forall X \in \Sigma$ 均存在相应的层级 $m \in \omega$ 使得 $\Gamma \Vdash X : m$ 成立时，判断 $\Gamma \mid \Sigma \vdash A : n$ 才是合法的（隐式前提）。

**组 1：基础结构规则**
$$ \text{(Axiom)} \quad \frac{A \in \Sigma \quad \Gamma \Vdash A : n}{\Gamma \mid \Sigma \vdash A : n} $$
$$ \text{(Arr-I)} \quad \frac{\Gamma \mid \Sigma \cup \{A\} \vdash B : n}{\Gamma \mid \Sigma \vdash \mathbf{arr}(A, B) : n} $$
$$ \text{(Arr-E)} \quad \frac{\Gamma \mid \Sigma \vdash \mathbf{arr}(A, B) : n \quad \Gamma \mid \Sigma \vdash A : n}{\Gamma \mid \Sigma \vdash B : n} $$

**组 2：同态映射的核心机制**
**规则 2.1 (Hom-Abs) - 跨层抽象：**
如果在一个扩展了 $n+1$ 层级变量的前提下，能证明底层蕴含关系，则可抽象为高层同态映射。
$$ \frac{(n+1) \cdot \Gamma \mid \uparrow_1^0(\Sigma) \cup \{A\} \vdash B : n}{\Gamma \mid \Sigma \vdash \mathbf{hom}(A, B) : n+1} $$

**规则 2.2 (Hom-Exec) - 降维执行：**
同态映射在其合法参数 $S$ 下的执行必须借助 $\mathbf{down}$ 进行。
$$ \frac{\Gamma \mid \Sigma \vdash \mathbf{hom}(A, B) : n+1 \quad \Gamma \Vdash S : n+1 \quad \Gamma \mid \Sigma \vdash \mathbf{down}(\mathbf{hom}(A, B), A[S]_0) : n}{\Gamma \mid \Sigma \vdash B[S]_0 : n} $$

**组 3：投射与结构分配律**
**规则 3.1 (Down-Act) - 投射生效：**
$$ \frac{\Gamma \mid \Sigma \vdash H : n+1 \quad \Gamma \mid \Sigma \vdash C : n}{\Gamma \mid \Sigma \vdash \mathbf{down}(H, C) : n} $$

**规则 3.2 (Hom-Dist-I) - 同态渗透（左到右）：**
同态映射作用于下层结构时，可以跨越逻辑联结词透射。
$$ \frac{\Gamma \mid \Sigma \vdash \mathbf{down}(H, \mathbf{arr}(X, Y)) : n}{\Gamma \mid \Sigma \vdash \mathbf{arr}(\mathbf{down}(H, X), \mathbf{down}(H, Y)) : n} $$

**规则 3.3 (Hom-Dist-E) - 同态渗透（右到左）：**
$$ \frac{\Gamma \mid \Sigma \vdash \mathbf{arr}(\mathbf{down}(H, X), \mathbf{down}(H, Y)) : n}{\Gamma \mid \Sigma \vdash \mathbf{down}(H, \mathbf{arr}(X, Y)) : n} $$


# 附录：跨层同态演算形式系统 ($\mathcal{H}_{CS}$) 核心元定理及证明

本附录基于 ZFC 集合论，严格确立系统 $\mathcal{H}_{CS}$ 的句法代数性质与层级类型系统的元定理。由于语法空间 $\mathbb{T}$ 定义为 $V_\omega$ 上严格单调算子 $\Phi$ 的最小不动点，根据 Knaster-Tarski 不动点定理，对 $\mathbb{T}$ 中元素的结构归纳法（Structural Induction）是严格良基且合法的。后续所有证明均直接依赖此归纳原理。对于 $\mathbf{bot}$, $\mathbf{arr}(A, B)$ 以及 $\mathbf{down}(H, A)$ 等不涉及 De Bruijn 索引变动的构造，其归纳步骤是平凡的同态分配，证明中将在展开核心绑定器 $\mathbf{hom}$ 时一并略写其显然步骤，以聚焦于变元捕获与指标偏移的本质。

## 附录 A：De Bruijn 结构代数引理

### 引理 A.1（代数升阶交换引理 / Lifting Exchange）
对于任意 $A \in \mathbb{T}$，以及 $c, k, d_1, d_2 \in \omega$，若 $c \le k$，则有：
$$ \uparrow_{d_2}^{k+d_1}(\uparrow_{d_1}^c(A)) = \uparrow_{d_1}^c(\uparrow_{d_2}^k(A)) $$

**证明：** 对 $A$ 进行结构归纳。
1. 若 $A = \mathbf{v}_i$：
   - 当 $i < c$ 时：由于 $c \le k$，必有 $i < k$ 且 $i < k+d_1$。
     左式 = $\uparrow_{d_2}^{k+d_1}(\mathbf{v}_i) = \mathbf{v}_i$。右式 = $\uparrow_{d_1}^c(\mathbf{v}_i) = \mathbf{v}_i$。等式成立。
   - 当 $c \le i < k$ 时：
     左式 = $\uparrow_{d_2}^{k+d_1}(\mathbf{v}_{i+d_1})$。因 $i < k$，有 $i+d_1 < k+d_1$，故左式 = $\mathbf{v}_{i+d_1}$。
     右式 = $\uparrow_{d_1}^c(\mathbf{v}_i)$。因 $i \ge c$，右式 = $\mathbf{v}_{i+d_1}$。等式成立。
   - 当 $i \ge k$ 时：必有 $i \ge c$。
     左式 = $\uparrow_{d_2}^{k+d_1}(\mathbf{v}_{i+d_1})$。因 $i \ge k$，有 $i+d_1 \ge k+d_1$，故左式 = $\mathbf{v}_{i+d_1+d_2}$。
     右式 = $\uparrow_{d_1}^c(\mathbf{v}_{i+d_2})$。因 $i \ge c$，有 $i+d_2 \ge c$，故右式 = $\mathbf{v}_{i+d_2+d_1}$。等式成立。
2. 若 $A = \mathbf{hom}(X, Y)$：
   左式 = $\uparrow_{d_2}^{k+d_1}(\mathbf{hom}(\uparrow_{d_1}^{c+1}(X), \uparrow_{d_1}^{c+1}(Y))) = \mathbf{hom}(\uparrow_{d_2}^{(k+1)+d_1}(\uparrow_{d_1}^{c+1}(X)), \uparrow_{d_2}^{(k+1)+d_1}(\uparrow_{d_1}^{c+1}(Y)))$。
   由归纳假设（此时截断值分别为 $c+1 \le k+1$），左式等于 $\mathbf{hom}(\uparrow_{d_1}^{c+1}(\uparrow_{d_2}^{k+1}(X)), \uparrow_{d_1}^{c+1}(\uparrow_{d_2}^{k+1}(Y)))$，这恰好是 $\uparrow_{d_1}^c(\uparrow_{d_2}^k(\mathbf{hom}(X, Y)))$ 的定义。
3. 其它结构（$\mathbf{bot}, \mathbf{arr}, \mathbf{down}$）直接由归纳假设得证。 $\blacksquare$

### 引理 A.2（代换抵消引理 / Cancellation）
对于任意 $A, S \in \mathbb{T}$ 及 $c \in \omega$，有：
$$ (\uparrow_1^c(A))[S]_c = A $$

**证明：** 对 $A$ 进行结构归纳。
1. 若 $A = \mathbf{v}_i$：
   - 当 $i < c$ 时：$(\uparrow_1^c(\mathbf{v}_i))[S]_c = \mathbf{v}_i[S]_c = \mathbf{v}_i = A$。
   - 当 $i \ge c$ 时：$(\uparrow_1^c(\mathbf{v}_i))[S]_c = \mathbf{v}_{i+1}[S]_c = \mathbf{v}_{(i+1)-1} = \mathbf{v}_i = A$。
2. 若 $A = \mathbf{hom}(X, Y)$：
   $(\uparrow_1^c(\mathbf{hom}(X,Y)))[S]_c = \mathbf{hom}(\uparrow_1^{c+1}(X), \uparrow_1^{c+1}(Y))[S]_c$
   $= \mathbf{hom}( (\uparrow_1^{c+1}(X))[\uparrow_1^0(S)]_{c+1}, (\uparrow_1^{c+1}(Y))[\uparrow_1^0(S)]_{c+1} )$。
   由归纳假设（代换目标为截断值 $c+1$），其等价于 $\mathbf{hom}(X, Y) = A$。
3. 其它结构由归纳假设同理可证。 $\blacksquare$

### 引理 A.3（升阶与代换分配引理 / Lifting over Substitution）
对于任意 $A, S \in \mathbb{T}$ 及 $c, k, d \in \omega$，若 $k \le c$，则有：
$$ \uparrow_d^k(A[S]_c) = (\uparrow_d^k(A))[\uparrow_d^k(S)]_{c+d} $$

**证明：** 对 $A$ 进行结构归纳。
1. 若 $A = \mathbf{v}_i$：
   - 当 $i < k \le c$ 时：$A[S]_c = \mathbf{v}_i$。左式 = $\uparrow_d^k(\mathbf{v}_i) = \mathbf{v}_i$。右式 = $\mathbf{v}_i[\dots]_{c+d} = \mathbf{v}_i$（因 $i < k \le c+d$）。成立。
   - 当 $k \le i < c$ 时：$A[S]_c = \mathbf{v}_i$。左式 = $\uparrow_d^k(\mathbf{v}_i) = \mathbf{v}_{i+d}$。右式 = $\mathbf{v}_{i+d}[\dots]_{c+d} = \mathbf{v}_{i+d}$（因 $i+d < c+d$）。成立。
   - 当 $i = c$ 时：$A[S]_c = S$。左式 = $\uparrow_d^k(S)$。右式 = $\uparrow_d^k(\mathbf{v}_c)[\uparrow_d^k(S)]_{c+d} = \mathbf{v}_{c+d}[\uparrow_d^k(S)]_{c+d} = \uparrow_d^k(S)$。成立。
   - 当 $i > c$ 时：$A[S]_c = \mathbf{v}_{i-1}$。因 $i > c \ge k$，有 $i-1 \ge k$。左式 = $\mathbf{v}_{i-1+d}$。右式 = $\mathbf{v}_{i+d}[\dots]_{c+d} = \mathbf{v}_{i+d-1}$（因 $i+d > c+d$）。成立。
2. 若 $A = \mathbf{hom}(X, Y)$：
   左式 = $\uparrow_d^k(\mathbf{hom}(X[\uparrow_1^0(S)]_{c+1}, \dots)) = \mathbf{hom}(\uparrow_d^{k+1}(X[\uparrow_1^0(S)]_{c+1}), \dots)$。
   因为 $k+1 \le c+1$，由对 $X$ 的归纳假设：
   内部项 = $(\uparrow_d^{k+1}(X))[\uparrow_d^{k+1}(\uparrow_1^0(S))]_{c+1+d}$。
   对于代入项 $\uparrow_d^{k+1}(\uparrow_1^0(S))$，由于 $0 \le k$，引用引理 A.1 有：$\uparrow_d^{k+1}(\uparrow_1^0(S)) = \uparrow_1^0(\uparrow_d^k(S))$。
   故内部项 = $(\uparrow_d^{k+1}(X))[\uparrow_1^0(\uparrow_d^k(S))]_{c+d+1}$。
   这正好等价于右式展开：$(\uparrow_d^k(\mathbf{hom}(X,Y)))[\uparrow_d^k(S)]_{c+d} = \mathbf{hom}(\uparrow_d^{k+1}(X), \dots)[\uparrow_d^k(S)]_{c+d}$。匹配成功。
3. 其它结构由归纳假设得证。 $\blacksquare$

### 引理 A.4（代换组合引理 / Substitution Composition）
对于任意 $A, S, T \in \mathbb{T}$ 及 $c, d \in \omega$，若 $c \le d$，则有：
$$ A[S]_c[T]_d = A[\uparrow_1^c(T)]_{d+1}[S[T]_d]_c $$

**证明：** 对 $A$ 进行结构归纳。
1. 若 $A = \mathbf{v}_i$：
   - 当 $i < c \le d$ 时：左式 = $\mathbf{v}_i[T]_d = \mathbf{v}_i$。右式 = $\mathbf{v}_i[\dots]_c = \mathbf{v}_i$。成立。
   - 当 $i = c$ 时：左式 = $S[T]_d$。右式 = $\mathbf{v}_c[S[T]_d]_c$（因 $c < d+1$） $= S[T]_d$。成立。
   - 当 $c < i \le d$ 时：左式 = $\mathbf{v}_{i-1}[T]_d = \mathbf{v}_{i-1}$。右式 = $\mathbf{v}_i[S[T]_d]_c = \mathbf{v}_{i-1}$。成立。
   - 当 $i = d+1$ 时：左式 = $\mathbf{v}_d[T]_d = T$。右式 = $(\uparrow_1^c(T))[S[T]_d]_c$。根据引理 A.2（抵消引理），该式直接等于 $T$。成立。
   - 当 $i > d+1$ 时：左式 = $\mathbf{v}_{i-1}[T]_d = \mathbf{v}_{i-2}$。右式 = $\mathbf{v}_{i-1}[S[T]_d]_c = \mathbf{v}_{i-2}$。成立。
2. 若 $A = \mathbf{hom}(X, Y)$：
   左式 = $\mathbf{hom}(X[\uparrow_1^0(S)]_{c+1}[\uparrow_1^0(T)]_{d+1}, \dots)$。
   因为 $c+1 \le d+1$，由归纳假设，其内部项等于：$X[\uparrow_1^{c+1}(\uparrow_1^0(T))]_{d+2}[ \uparrow_1^0(S)[\uparrow_1^0(T)]_{d+1} ]_{c+1}$。
   右式 = $\mathbf{hom}( X[\uparrow_1^0(\uparrow_1^c(T))]_{d+2} [ \uparrow_1^0(S[T]_d) ]_{c+1}, \dots )$。
   对比左右两式的两个代入项：
   - 第一代入项：$\uparrow_1^{c+1}(\uparrow_1^0(T))$ vs $\uparrow_1^0(\uparrow_1^c(T))$。由引理 A.1（令 $k=c, d_1=d_2=1$），二者严格相等。
   - 第二代入项：$\uparrow_1^0(S)[\uparrow_1^0(T)]_{d+1}$ vs $\uparrow_1^0(S[T]_d)$。由引理 A.3（令 $k=0, d=1$，其中 $c$ 对应此处截断 $d$），二者严格相等。
   故左右结构完全同构，等式成立。
3. 其它结构由归纳假设得证。 $\blacksquare$

## 附录 B：层级良构语义的元定理

定义算子 $\Gamma \oplus_k x$ 表示在上下文 $\Gamma$ 的索引 $k$ 处插入层级 $x \in \omega$。形式化地：
$$ (\Gamma \oplus_k x)(i) = \begin{cases} \Gamma(i), & \text{if } i < k \\ x, & \text{if } i = k \\ \Gamma(i-1), & \text{if } i > k \end{cases} $$
同时定义 $\Gamma_{\setminus c}$ 表示移除索引 $c$ 处的层级约束：
$$ \Gamma_{\setminus c}(i) = \begin{cases} \Gamma(i), & \text{if } i < c \\ \Gamma(i+1), & \text{if } i \ge c \end{cases} $$

### 引理 B.1（层级上下文弱化引理 / Context Weakening）
若推导式 $\Gamma \Vdash A : n$ 成立，则对任意 $k, x \in \omega$，有 $\Gamma \oplus_k x \Vdash \uparrow_1^k(A) : n$ 成立。

**证明：** 对 $\Gamma \Vdash A : n$ 的推导树进行数学归纳。
1. **(WF-Var)**: $A = \mathbf{v}_i$ 且 $\Gamma(i) = n$。
   - 若 $i < k$：$\uparrow_1^k(\mathbf{v}_i) = \mathbf{v}_i$。且 $(\Gamma \oplus_k x)(i) = \Gamma(i) = n$。故 $\Gamma \oplus_k x \Vdash \mathbf{v}_i : n$ 成立。
   - 若 $i \ge k$：$\uparrow_1^k(\mathbf{v}_i) = \mathbf{v}_{i+1}$。且 $(\Gamma \oplus_k x)(i+1) = \Gamma(i) = n$。故 $\Gamma \oplus_k x \Vdash \mathbf{v}_{i+1} : n$ 成立。
2. **(WF-Bot)** 和 **(WF-Arr)**: 由归纳假设和定义直接得证。
3. **(WF-Hom)**: $A = \mathbf{hom}(X, Y)$，且有前提 $(n+1) \cdot \Gamma \Vdash X : n$。
   注意 $(n+1) \cdot (\Gamma \oplus_k x)$ 等同于在上下文头部增加 $n+1$ 后，在偏移后的索引 $k+1$ 处插入 $x$，即：$((n+1) \cdot \Gamma) \oplus_{k+1} x$。
   根据归纳假设，在 $((n+1) \cdot \Gamma) \oplus_{k+1} x$ 下，$\uparrow_1^{k+1}(X) : n$ 成立。对 $Y$ 同理。
   应用 (WF-Hom) 规则，得到 $\Gamma \oplus_k x \Vdash \mathbf{hom}(\uparrow_1^{k+1}(X), \uparrow_1^{k+1}(Y)) : n+1$。
   其左侧正好是定义 2.1 中的 $\uparrow_1^k(\mathbf{hom}(X, Y))$，结论成立。
4. **(WF-Down)**: $A = \mathbf{down}(H, X)$。
   前提为 $\Gamma \Vdash H : n+1$ 与 $\Gamma \Vdash X : n$。由归纳假设得 $\Gamma \oplus_k x \Vdash \uparrow_1^k(H) : n+1$ 与 $\Gamma \oplus_k x \Vdash \uparrow_1^k(X) : n$。应用规则直接得出结论。 $\blacksquare$

### 定理 B.2（类型保持定理 / Typing Preservation under Substitution）
设 $\Gamma$ 为任意合法上下文，$c \in \omega$。若存在目标项 $S$ 满足 $\Gamma_{\setminus c} \Vdash S : \Gamma(c)$，则对于任何项 $A$，若 $\Gamma \Vdash A : n$，必有 $\Gamma_{\setminus c} \Vdash A[S]_c : n$。

*(注：该定理保证了跨层演绎系统在执行 (Hom-Exec) 及其它带有降维计算的逻辑步骤时，不会导致底层层级类型的崩溃。)*

**证明：** 对 $\Gamma \Vdash A : n$ 的推导树进行数学归纳。
1. **(WF-Var)**: $A = \mathbf{v}_i$ 且 $\Gamma(i) = n$。
   - 当 $i < c$ 时：$A[S]_c = \mathbf{v}_i$。此时 $\Gamma_{\setminus c}(i) = \Gamma(i) = n$。由 (WF-Var) 得 $\Gamma_{\setminus c} \Vdash \mathbf{v}_i : n$。
   - 当 $i = c$ 时：$A[S]_c = S$。此时前提 $\Gamma(c) = n$。根据给定的全局前提，$\Gamma_{\setminus c} \Vdash S : \Gamma(c)$，即 $\Gamma_{\setminus c} \Vdash S : n$ 成立。
   - 当 $i > c$ 时：$A[S]_c = \mathbf{v}_{i-1}$。此时 $\Gamma_{\setminus c}(i-1) = \Gamma(i) = n$。由 (WF-Var) 得 $\Gamma_{\setminus c} \Vdash \mathbf{v}_{i-1} : n$。
2. **(WF-Bot)** 和 **(WF-Arr)**: 根据结构的分配律，由归纳假设平凡可证。
3. **(WF-Hom)**: $A = \mathbf{hom}(X, Y)$，层级为 $n+1$。
   由前提推导可知：$(n+1) \cdot \Gamma \Vdash X : n$ 且 $(n+1) \cdot \Gamma \Vdash Y : n$。
   我们需要证明：$\Gamma_{\setminus c} \Vdash \mathbf{hom}(X, Y)[S]_c : n+1$，即 $\Gamma_{\setminus c} \Vdash \mathbf{hom}(X[\uparrow_1^0(S)]_{c+1}, Y[\uparrow_1^0(S)]_{c+1}) : n+1$。
   这要求证明：$(n+1) \cdot \Gamma_{\setminus c} \Vdash X[\uparrow_1^0(S)]_{c+1} : n$。
   令 $\Gamma' = (n+1) \cdot \Gamma$。显然有 $\Gamma'_{\setminus (c+1)} = (n+1) \cdot \Gamma_{\setminus c}$，且目标截断索引为 $c+1$。
   要应用对 $X$ 的归纳假设，必须满足代入项的类型约束：$\Gamma'_{\setminus (c+1)} \Vdash \uparrow_1^0(S) : \Gamma'(c+1)$。
   已知 $\Gamma'(c+1) = \Gamma(c)$。我们需要证明：$(n+1) \cdot \Gamma_{\setminus c} \Vdash \uparrow_1^0(S) : \Gamma(c)$。
   由全局前提已知 $\Gamma_{\setminus c} \Vdash S : \Gamma(c)$。根据**引理 B.1 (弱化引理)**，在上下文头部（索引 $0$）插入层级 $n+1$ 产生：
   $(\Gamma_{\setminus c}) \oplus_0 (n+1) \Vdash \uparrow_1^0(S) : \Gamma(c)$。
   因为 $(\Gamma_{\setminus c}) \oplus_0 (n+1)$ 正是 $(n+1) \cdot \Gamma_{\setminus c}$，故代换类型前提满足。
   应用归纳假设于 $X$：$(n+1) \cdot \Gamma_{\setminus c} \Vdash X[\uparrow_1^0(S)]_{c+1} : n$ 成立。（对 $Y$ 证明完全相同）
   应用 (WF-Hom) 规则，定理结论得证。
4. **(WF-Down)**: $A = \mathbf{down}(H, X)$。
   前提为 $\Gamma \Vdash H : n+1$ 且 $\Gamma \Vdash X : n$。
   由归纳假设分别得 $\Gamma_{\setminus c} \Vdash H[S]_c : n+1$ 与 $\Gamma_{\setminus c} \Vdash X[S]_c : n$。
   应用 (WF-Down) 规则直接得出 $\Gamma_{\setminus c} \Vdash \mathbf{down}(H[S]_c, X[S]_c) : n$，即等价于 $\Gamma_{\setminus c} \Vdash \mathbf{down}(H, X)[S]_c : n$。 $\blacksquare$
