
#### **1. 语言 (Language)**

本系统的形式语言 **L** 是一种含等词的一阶谓词逻辑语言。

1.  **非逻辑符号:**
    *   一个三元谓词符号：`P`

2.  **逻辑符号:**
    *   二元谓词符号：` = `
    *   命题联结词： `¬`, `→`
    *   量词： `∀`
    *   变量符号： `v₀, v₁, v₂, ...`
    *   辅助符号： `(`, `)`

#### **2. 构造规则 (Formation Rules)**

1.  **项 (Terms):** 本语言中的项仅包含变量 `v₀, v₁, v₂, ...`。
2.  **原子公式 (Atomic Formulas):** 若 `t₁, t₂, t₃` 为项，则 `t₁ = t₂` 与 `P(t₁, t₂, t₃)` 是原子公式。
3.  **合式公式 (Well-Formed Formulas - WFFs):**
    *   所有原子公式都是合式公式。
    *   若 `φ` 和 `ψ` 是合式公式，则 `(¬φ)` 和 `(φ → ψ)` 是合式公式。
    *   若 `φ` 是合式公式且 `x` 是一个变量，则 `(∀x φ)` 是合式公式。

#### **3. 定义 (Definitions)**

以下为标准逻辑符号的缩写：
*   `φ ∧ ψ`   :=   `¬(φ → ¬ψ)`
*   `φ ∨ ψ`   :=   `(¬φ) → ψ`
*   `φ ↔ ψ`   :=   `(φ → ψ) ∧ (ψ → φ)`
*   `∃x φ`    :=   `¬∀x (¬φ)`
*   `∃!x φ`   :=   `∃x (φ ∧ ∀y (φ[y/x] → y=x))`

以下为本系统的特有定义：

**D1: IsFunction(f)**
`∀x ∃!y P(f, x, y)`

#### **4. 公理系统 (Axiom System)**

本系统包含标准的一阶逻辑公理模式与等词公理。

1.  **逻辑公理:**
    *   所有重言式实例。
    *   `∀x φ(x) → φ(t)` (当项 `t` 在 `φ(x)` 中对 `x` 可代入时)。
    *   `∀x (φ → ψ) → (∀x φ → ∀x ψ)`
    *   `φ → ∀x φ` (当 `x` 在 `φ` 中不是自由变量时)。

2.  **等词公理:**
    *   `∀x (x=x)`
    *   `∀x ∀y (x=y → (φ(x) → φ(y)))`

3.  **特有公理:**

    **A1: Uniqueness**
    `∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`

    **A2: Extensionality**
    `∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`

	**A3: Axiom of Internalized Application and Commutativity**
	`∃p ( IsFunction(p) ∧ ∀f ( IsFunction(f) → (∀y (P(p, f, y) ↔ P(f, p, y))) ) )`

	**A4: Axiom of Involutive Negation**
	`∃n ( IsFunction(n) ∧ ∀x (∀y (P(n, x, y) → ∀z (P(n, y, z) → z=x))) ∧ ∀x (∀y (P(n, x, y) → x≠y)) )`


#### **5. 推理规则 (Rules of Inference)**

1.  **分离规则 (Modus Ponens):**
    从 `φ` 和 `φ → ψ`，可推出 `ψ`。

2.  **概括规则 (Generalization):**
    从 `φ`，可推出 `∀x φ`。

### **6. 模型 (Model)**

令 **M** 为一个 **L**-结构，其定义如下：

`M = ⟨U, Pᴹ⟩`

#### **6.1. 论域 (Universe)**

模型的论域 `U` 是一个包含四个不同元素的集合：

`U = {T, F, N, I}`

#### **6.2. 谓词的解释 (Interpretation of the Predicate)**

三元谓词符号 `P` 的解释 `Pᴹ` 是 `U × U × U` 的一个子集，定义如下：

`Pᴹ = {`
`(I, T, T), (I, F, F), (I, N, N), (I, I, I),`
`(N, T, F), (N, F, T), (N, N, I), (N, I, N)`
`}`

对于任何不在此集合中的三元组 `(a, b, c)`，`Pᴹ(a, b, c)` 为假。

#### **6.3. 公理验证 (Axiom Verification)**

我们验证结构 **M** 满足公理 A1, A2, A3, A4。

**A1: Uniqueness**
`∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`

*   **验证:** 我们通过检查 `Pᴹ` 的定义来验证。
    *   若 `a ∈ {T, F}`，则对于任何 `b, c₁`，`Pᴹ(a, b, c₁)` 为假，因此蕴含式的前提为假，整个公式为真。
    *   若 `a = I`，对于每一个 `b ∈ U`，都存在唯一的 `c` 使得 `Pᴹ(I, b, c)` 为真（即 `c=b`）。
    *   若 `a = N`，对于每一个 `b ∈ U`，都存在唯一的 `c` 使得 `Pᴹ(N, b, c)` 为真。
    因此，对于任何 `a, b ∈ U`，至多只有一个 `c` 使得 `Pᴹ(a, b, c)` 为真。故公理 **A1** 在 **M** 中成立。

**A2: Extensionality**
`∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`

*   **验证:**
    *   `→` 方向 (`a = b → ...`) 是等词公理 `∀x ∀y (x=y → (φ(x) → φ(y)))` 的一个直接实例，因此在任何标准一阶逻辑模型中都成立。
    *   `←` 方向 (`(...) → a = b`)，我们通过其等价的逆否命题来验证：对于任意 `a, b ∈ U`，若 `a ≠ b`，则它们在谓词 `P` 中的行为是可区分的。即，存在 `x, y ∈ U` 使得 `(P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b))` 为假。
    
    我们需要对论域 `U = {T, F, N, I}` 中所有 `C(4, 2) = 6` 对不同的元素进行检验。

    1.  **区分 `T` 和 `F`:**
        令 `x = I`, `y = T`。
        `Pᴹ(I, T, T)` 为真，而 `Pᴹ(I, T, F)` 为假。
        因此 `P(x, y, T) ↔ P(x, y, F)` 为假，这使得 `∀x ∀y (...)` 的整个合取式为假。

    2.  **区分 `T` 和 `N`:**
        令 `x = T`, `y = F`。
        `Pᴹ(T, T, F)` 为假，而 `Pᴹ(N, T, F)` 为真。
        因此 `P(T, x, y) ↔ P(N, x, y)` 为假。

    3.  **区分 `T` 和 `I`:**
        令 `x = T`, `y = T`。
        `Pᴹ(T, T, T)` 为假，而 `Pᴹ(I, T, T)` 为真。
        因此 `P(T, x, y) ↔ P(I, x, y)` 为假。

    4.  **区分 `F` 和 `N`:**
        令 `x = F`, `y = T`。
        `Pᴹ(F, F, T)` 为假，而 `Pᴹ(N, F, T)` 为真。
        因此 `P(F, x, y) ↔ P(N, x, y)` 为假。

    5.  **区分 `F` 和 `I`:**
        令 `x = F`, `y = F`。
        `Pᴹ(F, F, F)` 为假，而 `Pᴹ(I, F, F)` 为真。
        因此 `P(F, x, y) ↔ P(I, x, y)` 为假。

    6.  **区分 `N` 和 `I`:**
        令 `x = T`, `y = F`。
        `Pᴹ(N, T, F)` 为真，而 `Pᴹ(I, T, F)` 为假。
        因此 `P(N, x, y) ↔ P(I, x, y)` 为假。

    由于任意两个不同的元素 `a, b ∈ U` 在 `P` 关系中的行为都是可区分的，`←` 方向的蕴含式成立。
    
    因此，公理 **A2** 在 **M** 中成立。

**A3: Axiom of Internalized Application and Commutativity**
`∃p ( IsFunction(p) ∧ ∀f ( IsFunction(f) → (∀y (P(p, f, y) ↔ P(f, p, y))) ) )`

*   **验证:** 我们选择 `p` 的赋值为 `I`。
    1.  **验证 `M ⊨ IsFunction(I)`:**
        `IsFunction(I)` 即 `∀x ∃!y P(I, x, y)`。对于 `U` 中任意的 `x` (`T, F, N, I`)，根据 `Pᴹ` 的定义，存在唯一的 `y=x` 使得 `Pᴹ(I, x, y)` 为真。故 `I` 是一个函数。
    2.  **验证 `M ⊨ ∀f ( IsFunction(f) → (∀y (P(I, f, y) ↔ P(f, I, y))) )`**:
        首先，确定 `U` 中哪些元素 `f` 满足 `IsFunction(f)`。
        *   `f=I` 满足 (已证)。
        *   `f=N`: 对于 `U` 中任意的 `x`，都存在唯一的 `y` 使得 `Pᴹ(N, x, y)` 为真。故 `N` 是一个函数。
        *   `f=T` 或 `f=F`: 对于任意 `x` 都不存在 `y` 使得 `Pᴹ(T,x,y)` 或 `Pᴹ(F,x,y)` 为真，故它们不是函数。
        所以，我们需要对 `f=I` 和 `f=N` 检验该蕴含式。
        *   **情形 f = I:** `∀y (P(I, I, y) ↔ P(I, I, y))`。此为重言式，成立。
        *   **情形 f = N:** `∀y (P(I, N, y) ↔ P(N, I, y))`。`Pᴹ(I, N, y)` 仅在 `y=N` 时为真。`Pᴹ(N, I, y)` 仅在 `y=N` 时为真。因为 `y=N ↔ y=N` 对所有 `y` 成立，所以该式成立。
    因为存在元素 `p=I` 满足所有条件，故公理 **A3** 在 **M** 中成立。

**A4: Axiom of Involutive Negation**
`∃n ( IsFunction(n) ∧ ∀x (∀y (P(n, x, y) → ∀z (P(n, y, z) → z=x))) ∧ ∀x (∀y (P(n, x, y) → x≠y)) )`

*   **验证:** 我们选择 `n` 的赋值为 `N`。
    1.  **验证 `M ⊨ IsFunction(N)`:** 在 A3 的验证中已经证明，成立。
    2.  **验证 `M ⊨ ∀x (∀y (P(N, x, y) → ∀z (P(N, y, z) → z=x)))`:** (即 `N(N(x)) = x`)
        *   `x=T`: `Pᴹ(N, T, F)` 为真。则需验证 `∀z (P(N, F, z) → z=T)`。`Pᴹ(N, F, z)` 仅在 `z=T` 时为真，`T=T` 成立。
        *   `x=F`: `Pᴹ(N, F, T)` 为真。则需验证 `∀z (P(N, T, z) → z=F)`。`Pᴹ(N, T, z)` 仅在 `z=F` 时为真，`F=F` 成立。
        *   `x=N`: `Pᴹ(N, N, I)` 为真。则需验证 `∀z (P(N, I, z) → z=N)`。`Pᴹ(N, I, z)` 仅在 `z=N` 时为真，`N=N` 成立。
        *   `x=I`: `Pᴹ(N, I, N)` 为真。则需验证 `∀z (P(N, N, z) → z=I)`。`Pᴹ(N, N, z)` 仅在 `z=I` 时为真，`I=I` 成立。
        该条件对所有 `x ∈ U` 成立。
    3.  **验证 `M ⊨ ∀x (∀y (P(N, x, y) → x≠y))`:** (即 `N(x) ≠ x`)
        *   `x=T`: `Pᴹ(N, T, F)` 为真，需验证 `T ≠ F`。成立。
        *   `x=F`: `Pᴹ(N, F, T)` 为真，需验证 `F ≠ T`。成立。
        *   `x=N`: `Pᴹ(N, N, I)` 为真，需验证 `N ≠ I`。成立。
        *   `x=I`: `Pᴹ(N, I, N)` 为真，需验证 `I ≠ N`。成立。
        该条件对所有 `x ∈ U` 成立。
    因为存在元素 `n=N` 满足所有条件，故公理 **A4** 在 **M** 中成立。

由于所有公理都在 **M** 中成立，**M** 是该形式系统的一个模型。由于 `|U|=4 > 1`，它是一个非平凡模型。
