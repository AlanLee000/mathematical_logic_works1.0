#### **1. 语言 (Language)**

本系统的形式语言 **L** 是一种含等词的一阶谓词逻辑语言。

1.  **非逻辑符号:**
    *   一个三元谓词符号：`P`

2.  **逻辑符号:**
    *   二元谓词符号：` = `
    *   命题联结词： `¬`, `→`
    *   量词： `∀`
    *   变量符号： `v₀, v₁, v₂, ...`
    *   辅助符号： `(`, `)`

#### **2. 构造规则 (Formation Rules)**

1.  **项 (Terms):** 本语言中的项仅包含变量 `v₀, v₁, v₂, ...`。
2.  **原子公式 (Atomic Formulas):** 若 `t₁, t₂, t₃` 为项，则 `t₁ = t₂` 与 `P(t₁, t₂, t₃)` 是原子公式。
3.  **合式公式 (Well-Formed Formulas - WFFs):**
    *   所有原子公式都是合式公式。
    *   若 `φ` 和 `ψ` 是合式公式，则 `(¬φ)` 和 `(φ → ψ)` 是合式公式。
    *   若 `φ` 是合式公式且 `x` 是一个变量，则 `(∀x φ)` 是合式公式。

#### **3. 定义 (Definitions)**

以下为标准逻辑符号的缩写：
*   `φ ∧ ψ`   :=   `¬(φ → ¬ψ)`
*   `φ ∨ ψ`   :=   `(¬φ) → ψ`
*   `φ ↔ ψ`   :=   `(φ → ψ) ∧ (ψ → φ)`
*   `∃x φ`    :=   `¬∀x (¬φ)`
*   `∃!x φ`   :=   `∃x (φ ∧ ∀y (φ[y/x] → y=x))`

以下为本系统的特有定义：

**D1: IsFunction(f)**
`∀x ∃!y P(f, x, y)`

**D2: IsOperator(a)**
`∃x ∃y P(a, x, y)`

**D3: IsTerminus(t)**
`¬IsOperator(t)`

**D4: IsComplementor(N)**
`IsFunction(N) ∧ ∀x ( (IsTerminus(x) → ∀y (P(N, x, y) → IsOperator(y))) ∧ (IsOperator(x) → ∀y (P(N, x, y) → IsTerminus(y))) )`

**D5: IsInvolution(N)**
`∀x ∀y (P(N, x, y) → P(N, y, x))`

#### **4. 公理系统 (Axiom System)**

本系统包含标准的一阶逻辑公理模式与等词公理。

1.  **逻辑公理:**
    *   所有重言式实例。
    *   `∀x φ(x) → φ(t)` (当项 `t` 在 `φ(x)` 中对 `x` 可代入时)。
    *   `∀x (φ → ψ) → (∀x φ → ∀x ψ)`
    *   `φ → ∀x φ` (当 `x` 在 `φ` 中不是自由变量时)。

2.  **等词公理:**
    *   `∀x (x=x)`
    *   `∀x ∀y (x=y → (φ(x) → φ(y)))`

3.  **特有公理:**

    **A1: Uniqueness**
    `∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`

    **A2: Extensionality**
    `∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`

    **A3: Restricted Conjugation**
    `∀f ( IsFunction(f) → ∃g (IsFunction(g) ∧ ∀a ( IsOperator(a) → ∀b (P(f, a, b) ↔ P(a, g, b)) ) ) )`

    **A4: Existence of Complementor**
    `∃!N IsComplementor(N)`

    **A5: Complementor is Involutionary**
    `∀N (IsComplementor(N) → IsInvolution(N))`

#### **5. 推理规则 (Rules of Inference)**

1.  **分离规则 (Modus Ponens):**
    从 `φ` 和 `φ → ψ`，可推出 `ψ`。

2.  **概括规则 (Generalization):**
    从 `φ`，可推出 `∀x φ`。


### **6. 模型、可满足性与非平凡性**

设 **T** 为由特有公理 A1-A5 构成的公理集。本节将构造一个 **L**-结构 **M**，并证明 **M** 是 **T** 的一个非平凡模型，即 **M** ⊨ **T**。

#### **6.1 模型构造 (Model Construction)**

定义 **L**-结构 **M** = ⟨ D, Pᴹ ⟩ 如下：

1.  **论域 (Domain):** D = { o, t }
2.  **谓词 `P` 的释义 (Interpretation):** Pᴹ ⊂ D × D × D 定义为：
    Pᴹ = { (o, o, t), (o, t, o) }

#### **6.2 定义谓词的释义 (Interpretation of Defined Predicates)**

在模型 **M** 中，我们首先确定由定义 D1-D5 所引入谓词的真值。

1.  **IsOperator(a) := ∃x ∃y P(a, x, y)**
    *   **a = o:**  M ⊨ ∃x ∃y P(o, x, y) 为真，因为存在赋值 x=o, y=t 使得 (o, o, t) ∈ Pᴹ。因此 **M ⊨ IsOperator(o)**。
    *   **a = t:**  M ⊨ ∃x ∃y P(t, x, y) 为假，因为对于任意 x, y ∈ D，(t, x, y) ∉ Pᴹ。因此 **M ⊨ ¬IsOperator(t)**。

2.  **IsTerminus(a) := ¬IsOperator(a)**
    *   **a = o:** 由上可知，M ⊨ ¬IsOperator(o) 为假。因此 **M ⊨ ¬IsTerminus(o)**。
    *   **a = t:** 由上可知，M ⊨ ¬IsOperator(t) 为真。因此 **M ⊨ IsTerminus(t)**。

3.  **IsFunction(f) := ∀x ∃!y P(f, x, y)**
    *   **f = o:**
        *   **x = o:** P(o, o, y) 为真当且仅当 y=t。存在且唯一的 y 成立。
        *   **x = t:** P(o, t, y) 为真当且仅当 y=o。存在且唯一的 y 成立。
        *   由于对所有 x ∈ D 均成立，故 **M ⊨ IsFunction(o)**。
    *   **f = t:**
        *   **x = o:** 不存在任何 y ∈ D 使得 P(t, o, y) 为真。因此 ∃y P(t, o, y) 为假。
        *   故 M ⊨ ∀x ∃!y P(t, x, y) 为假。因此 **M ⊨ ¬IsFunction(t)**。

4.  **IsComplementor(N) := IsFunction(N) ∧ ∀x ( (IsTerminus(x) → ...) ∧ (IsOperator(x) → ...) )**
    *   **N = t:** 由于 M ⊨ ¬IsFunction(t)，合取式为假。因此 **M ⊨ ¬IsComplementor(t)**。
    *   **N = o:**
        *   第一部分：M ⊨ IsFunction(o) 为真。
        *   第二部分：∀x (...)
            *   **x = t:** M ⊨ IsTerminus(t)。需验证 ∀y (P(o, t, y) → IsOperator(y))。P(o, t, y) 仅在 y=o 时为真。此时 M ⊨ IsOperator(o) 为真。蕴含式成立。
            *   **x = o:** M ⊨ IsOperator(o)。需验证 ∀y (P(o, o, y) → IsTerminus(y))。P(o, o, y) 仅在 y=t 时为真。此时 M ⊨ IsTerminus(t) 为真。蕴含式成立。
        *   由于合取的两部分均为真，故 **M ⊨ IsComplementor(o)**。

5.  **IsInvolution(N) := ∀x ∀y (P(N, x, y) → P(N, y, x))**
    *   **N = t:** 对于任意 x, y ∈ D，P(t, x, y) 为假，故蕴含式 (¬P(t, x, y) ∨ P(t, y, x)) 恒为真。因此 **M ⊨ IsInvolution(t)**。
    *   **N = o:**
        *   **x=o, y=t:** P(o, o, t) 为真。P(o, t, o) 也为真。蕴含式成立。
        *   **x=t, y=o:** P(o, t, o) 为真。P(o, o, t) 也为真。蕴含式成立。
        *   对于所有其他 (x,y) 的组合，P(o, x, y) 为假，蕴含式恒为真。
        *   因此 **M ⊨ IsInvolution(o)**。

#### **6.3 可满足性证明**

我们逐一证明模型 **M** 满足公理 A1-A5。

*   **证明 M ⊨ A1: Uniqueness**
    `∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`
    1.  若 a≠o，则 P(a, b, c) 恒为假，蕴含式前件为假，公式为真。
    2.  若 a=o:
        *   若 b=o: P(o, o, c) 为真当且仅当 c=t。因此 c₁=t 且 c₂=t，故 c₁=c₂。
        *   若 b=t: P(o, t, c) 为真当且仅当 c=o。因此 c₁=o 且 c₂=o，故 c₁=c₂。
    因此，对于 D 中所有元素，公理 A1 成立。

*   **证明 M ⊨ A2: Extensionality**
    `∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a,x,y)↔P(b,x,y)) ∧ (P(x,a,y)↔P(x,b,y)) ∧ (P(x,y,a)↔P(x,y,b)) ) ) )`
    `→` 方向：为等词公理的直接推论，在任何标准模型中均成立。
    `←` 方向：需证明若 a≠b，则右侧合取式为假。令 a=o, b=t。
    1.  考虑子句 `∀x ∀y (P(o, x, y) ↔ P(t, x, y))`。
    2.  令 x=o, y=t。M ⊨ P(o, o, t) 为真，但 M ⊨ P(t, o, t) 为假。
    3.  因此 `真 ↔ 假` 为假。该子句为假。
    4.  整个合取式为假。故 a≠b → ¬(...) 为真。
    因此，公理 A2 成立。

*   **证明 M ⊨ A3: Restricted Conjugation**
    `∀f ( IsFunction(f) → ∃g (IsFunction(g) ∧ ∀a ( IsOperator(a) → ∀b (P(f, a, b) ↔ P(a, g, b)) ) ) )`
    1.  根据 6.2.3，唯一满足 IsFunction(f) 的元素是 f=o。我们只需验证 f=o 的情况。
    2.  需证明 M ⊨ `∃g (IsFunction(g) ∧ ∀a ( IsOperator(a) → ∀b (P(o, a, b) ↔ P(a, g, b)) ) )`。
    3.  令 g=o。根据 6.2.3，M ⊨ IsFunction(o) 为真。
    4.  根据 6.2.1，唯一满足 IsOperator(a) 的元素是 a=o。我们只需验证 a=o 的情况。
    5.  公式简化为验证 `∀b (P(o, o, b) ↔ P(o, o, b))`。这是一个重言式，恒为真。
    6.  我们找到了满足条件的 g (即 o)，故存在量化公式为真。
    因此，公理 A3 成立。

*   **证明 M ⊨ A4: Existence of Complementor**
    `∃!N IsComplementor(N)`
    1.  根据 6.2.4，我们证明了 M ⊨ IsComplementor(o) 且 M ⊨ ¬IsComplementor(t)。
    2.  这表明在论域 D 中，存在一个元素 o 满足 IsComplementor(N)，且对于 D 中任何其他元素 t，都不满足 IsComplementor(N)。
    3.  这完全符合 `∃!N` 的语义。
    因此，公理 A4 成立。

*   **证明 M ⊨ A5: Complementor is Involutionary**
    `∀N (IsComplementor(N) → IsInvolution(N))`
    1.  根据 6.2.4，唯一满足 IsComplementor(N) 的元素是 N=o。我们只需验证 N=o 的情况。
    2.  需验证 `IsComplementor(o) → IsInvolution(o)`。
    3.  根据 6.2.4，M ⊨ IsComplementor(o) 为真。
    4.  根据 6.2.5，M ⊨ IsInvolution(o) 为真。
    5.  `真 → 真` 为真。
    因此，公理 A5 成立。

**结论:** 模型 **M** 满足公理集 **T** 中的所有公理。

#### **6.4 非平凡性证明 (Proof of Non-Triviality)**

模型 **M** 是非平凡的，基于以下两点：

1.  **论域基数 (Cardinality of the Domain):**
    模型 **M** 的论域 D = { o, t } 的基数为 |D|=2。由于 |D| > 1，该模型在论域上是非平凡的。

2.  **谓词释义 (Interpretation of Predicates):**
    *   **基本谓词 `P`:** Pᴹ = { (o, o, t), (o, t, o) }。Pᴹ 既非空集 (∅)，也非全集 (D×D×D)。因此，原子公式 `P(x,y,z)` 在模型中是可满足的但非普遍有效的。
    *   **定义谓词:** 谓词 `IsOperator(x)` 和 `IsTerminus(x)` 将论域 D 非平凡地划分为两个不相交的非空子集 {o} 和 {t}。这表明模型具有能够区分不同元素的内部结构。

综上所述，模型 **M** 是一个非平凡的模型。


### **7. 无限模型与扩展公理系统**

本节旨在通过引入一条新的公理来排除简单的有限模型，并为此扩展后的公理系统构造一个非退化的无限模型。设 **T** 为由公理 A1-A5 构成的集合。

#### **7.1 新公理 (Axiom A6)**

我们向公理系统 **T** 中加入以下公理：

**A6: 惰性算子公理 (Axiom of Inert Operators)**
`∃a ( IsOperator(a) ∧ (∀N(IsComplementor(N) → a ≠ N)) ∧ ∀t(IsTerminus(t) → ∀y(¬P(a, t, y))) )`

此公理断言，存在一个并非补元(Complementor)的算子(Operator)，该算子不对任何终点元(Terminus)进行操作。

设 **T'** = **T** ∪ {A6} 为扩展后的公理集。

#### **7.2 无限模型构造 (Infinite Model Construction)**

定义 **L**-结构 **M_∞** = ⟨ D_∞, P_∞ᴹ ⟩ 如下：

1.  **论域 (Domain):** D_∞ = `ℤ` (全体整数集合)。

2.  **谓词 `P` 的释义 (Interpretation):** P_∞ᴹ ⊂ D_∞ × D_∞ × D_∞ 定义为：
    P_∞ᴹ = { (a, b, c) ∈ ℤ³ | (a = 0 ∧ c = 1 - b) ∨ (a ∈ 2ℤ \ {0} ∧ b = 0 ∧ c = 1 - a) }
    其中 `2ℤ` 代表偶数集。

#### **7.3 定义谓词在 M_∞ 中的释义**

1.  **IsOperator(a):** `M_∞ ⊨ IsOperator(a)` 当且仅当 `a` 为偶数 (`a ∈ 2ℤ`)。
    *   **证明:** `IsOperator(a) := ∃x ∃y P(a, x, y)`.
        *   若 `a=0`，则取 `x=0, y=1`，有 `(0,0,1) ∈ P_∞ᴹ`。故 `M_∞ ⊨ IsOperator(0)`。
        *   若 `a` 为非零偶数，则取 `x=0, y=1-a`，有 `(a,0,1-a) ∈ P_∞ᴹ`。故 `M_∞ ⊨ IsOperator(a)`。
        *   若 `a` 为奇数，根据 P_∞ᴹ 的定义，`a` 不能作为第一个元素，故 `P(a,x,y)` 恒为假。因此 `M_∞ ⊨ ¬IsOperator(a)`。

2.  **IsTerminus(t):** `M_∞ ⊨ IsTerminus(t)` 当且仅当 `t` 为奇数 (`t ∉ 2ℤ`)。
    *   **证明:** `IsTerminus(t) := ¬IsOperator(t)`. 由上可知，这等价于 `t` 为奇数。

3.  **IsFunction(f):** `M_∞ ⊨ IsFunction(f)` 当且仅当 `f = 0`。
    *   **证明:** `IsFunction(f) := ∀x ∃!y P(f, x, y)`.
        *   若 `f=0`，对于任意 `x ∈ ℤ`，`P(0,x,y)` 为真当且仅当 `y=1-x`。此 `y` 存在且唯一。故 `M_∞ ⊨ IsFunction(0)`。
        *   若 `f` 为非零偶数，则 `P(f,x,y)` 仅在 `x=0` 时有解 `y=1-f`。对于 `x=1`，不存在 `y` 使得 `P(f,1,y)` 为真。因此 `∀x` 不成立。
        *   若 `f` 为奇数，`P(f,x,y)` 恒为假，`∃y` 不成立。
        *   综上，仅 `f=0` 满足 `IsFunction(f)`。

4.  **IsComplementor(N):** `M_∞ ⊨ IsComplementor(N)` 当且仅当 `N = 0`。
    *   **证明:** `IsComplementor(N) := IsFunction(N) ∧ ...`。
        *   由上可知，`IsFunction(N)` 仅在 `N=0` 时为真。我们只需验证 `N=0` 是否满足定义的后半部分。
        *   `∀x ( (IsTerminus(x) → ∀y (P(0, x, y) → IsOperator(y))) ∧ (IsOperator(x) → ∀y (P(0, x, y) → IsTerminus(y))) )`
            *   若 `x` 为 Terminus (奇数)，则 `P(0,x,y)` 蕴含 `y=1-x`。`1 - (奇数)` 为偶数，故 `y` 为 Operator。蕴含式成立。
            *   若 `x` 为 Operator (偶数)，则 `P(0,x,y)` 蕴含 `y=1-x`。`1 - (偶数)` 为奇数，故 `y` 为 Terminus。蕴含式成立。
        *   因此，`M_∞ ⊨ IsComplementor(0)`。由于 `0` 是唯一的 Function，故也是唯一的 Complementor。

5.  **IsInvolution(N):** `M_∞ ⊨ IsInvolution(0)` 为真。
    *   **证明:** `IsInvolution(0) := ∀x ∀y (P(0, x, y) → P(0, y, x))`.
        *   `P(0, x, y)` 为真，当且仅当 `y = 1 - x`。
        *   `P(0, y, x)` 为真，当且仅当 `x = 1 - y`。
        *   `y = 1 - x` 与 `x = 1 - y` 是等价的代数式。因此蕴含式 `P(0, x, y) → P(0, y, x)` 恒为真。

#### **7.4 扩展公理集 T' 的可满足性证明**

我们将证明 `M_∞ ⊨ T'`，即 `M_∞` 满足公理 A1-A6。

*   **证明 M_∞ ⊨ A1: Uniqueness**
    `∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`
    1.  若 `a=0`，`P(0, b, c)` 为真要求 `c = 1-b`。对于给定的 `b`，`c` 是唯一确定的。故 `c₁ = c₂`。
    2.  若 `a` 为非零偶数，`P(a, b, c)` 为真要求 `b=0` 且 `c=1-a`。对于给定的 `a` 和 `b=0`，`c` 是唯一确定的。故 `c₁ = c₂`。
    3.  若 `a` 为奇数，`P(a, b, c)` 恒为假，蕴含式前件为假，公式为真。
    因此，公理 A1 成立。

*   **证明 M_∞ ⊨ A2: Extensionality**
    `∀a ∀b ( a = b ↔ ( ... ) )`
    1.  `→` 方向：为等词公理的直接推论。
    2.  `←` 方向：需证明若 `a ≠ b`，则 `∀x ∀y ( (P(a,x,y)↔P(b,x,y)) ∧ ... )` 为假。
    3.  令 `a, b ∈ D_∞` 且 `a ≠ b`。
    4.  考虑 `P(x,y,z)` 在第一个位置 `x` 的情况。`∀x ∀y (P(a,x,y) ↔ P(b,x,y))` 必须为真。
    5.  若 `a=0, b≠0`。令 `x=1`，则 `P(0,1,0)` 为真，但 `P(b,1,0)` 为假（因 `b≠0` 时要求 `x` 必须为 `0`）。`真 ↔ 假` 为假。
    6.  若 `a, b` 均为非零偶数。令 `x=0`。`P(a,0,1-a)` 为真，但 `P(b,0,1-a)` 为假（因为它要求结果为 `1-b`，而 `1-b ≠ 1-a`）。`真 ↔ 假` 为假。
    7.  其他情况亦可类似证明。因此 `a ≠ b` 蕴含了右侧公式为假。
    因此，公理 A2 成立。

*   **证明 M_∞ ⊨ A3: Restricted Conjugation**
    `∀f ( IsFunction(f) → ∃g (IsFunction(g) ∧ ...) )`
    1.  在 `M_∞` 中，`IsFunction(f)` 仅当 `f=0` 时为真。我们只需验证 `f=0` 的情况。
    2.  需证明 `∃g (IsFunction(g) ∧ ∀a ( IsOperator(a) → ∀b (P(0, a, b) ↔ P(a, g, b)) ) )`。
    3.  令 `g=0`。已知 `M_∞ ⊨ IsFunction(0)`。
    4.  需验证 `∀a ( IsOperator(a) → ∀b (P(0, a, b) ↔ P(a, 0, b)) )`。
        *   `a` 是 Operator，即 `a` 是偶数。
        *   若 `a=0`，公式变为 `∀b (P(0, 0, b) ↔ P(0, 0, b))`，为重言式。
        *   若 `a` 为非零偶数，`P(0, a, b)` 为真 ⇔ `b = 1-a`。而 `P(a, 0, b)` 为真 ⇔ `b = 1-a`。两者等价。
    5.  我们找到了 `g=0` 满足条件，故存在量化为真。
    因此，公理 A3 成立。

*   **证明 M_∞ ⊨ A4: Existence of Complementor**
    `∃!N IsComplementor(N)`
    1.  根据 7.3.4，我们证明了 `M_∞ ⊨ IsComplementor(0)`。
    2.  并且对于任何 `N ≠ 0`，`M_∞ ⊨ ¬IsFunction(N)`，因此 `M_∞ ⊨ ¬IsComplementor(N)`。
    3.  这表明在论域 `D_∞` 中，存在且仅存在一个元素 `0` 满足 `IsComplementor`。
    因此，公理 A4 成立。

*   **证明 M_∞ ⊨ A5: Complementor is Involutionary**
    `∀N (IsComplementor(N) → IsInvolution(N))`
    1.  在 `M_∞` 中，前件 `IsComplementor(N)` 仅在 `N=0` 时为真。
    2.  根据 7.3.5，我们证明了 `M_∞ ⊨ IsInvolution(0)` 为真。
    3.  `真 → 真` 为真。
    因此，公理 A5 成立。

*   **证明 M_∞ ⊨ A6: Axiom of Inert Operators**
    `∃a ( IsOperator(a) ∧ (∀N(IsComplementor(N) → a ≠ N)) ∧ ∀t(IsTerminus(t) → ∀y(¬P(a, t, y))) )`
    1.  在 `M_∞` 中，唯一的 Complementor `N` 是 `0`。
    2.  令 `a = 2`。
    3.  验证合取式的各部分：
        *   `IsOperator(2)`: `2` 是偶数，为真。
        *   `∀N(IsComplementor(N) → 2 ≠ N)`: `N` 只能是 `0`，而 `2 ≠ 0`，为真。
        *   `∀t(IsTerminus(t) → ∀y(¬P(2, t, y)))`:
            *   `t` 是 Terminus，即 `t` 是奇数。
            *   根据 P_∞ᴹ 定义，`P(2, b, c)` 为真要求 `b=0`。
            *   由于 `t` 是奇数，所以 `t ≠ 0`。
            *   因此，`P(2, t, y)` 对任何奇数 `t` 和任何 `y` 恒为假。
            *   故 `¬P(2, t, y)` 恒为真，蕴含式成立。
    4.  存在元素 `a=2` 满足所有条件，故存在量化为真。
    因此，公理 A6 成立。

**结论:** 无限模型 **M_∞** 满足扩展公理集 **T'**。

#### **7.5 原始模型的非相容性**

我们证明在 6.1 节中构造的有限模型 **M** = ⟨ { o, t }, { (o, o, t), (o, t, o) } ⟩ 不满足公理 A6。

*   **证明 $M \not \models A6$**
    1.  公理为 `∃a ( IsOperator(a) ∧ (∀N(IsComplementor(N) → a ≠ N)) ∧ ∀t(...) )`。
    2.  在模型 **M** 中，根据 6.2 节的分析：
        *   `IsOperator(a)` 为真当且仅当 `a=o`。
        *   `IsComplementor(N)` 为真当且仅当 `N=o`。
    3.  我们检验论域 `D = {o, t}` 中的每一个元素 `a` 是否能使存在量化为真。
    4.  **情况 1: a = t**
        *   `IsOperator(t)` 为假。合取式的第一项为假，故 `a=t` 不满足条件。
    5.  **情况 2: a = o**
        *   `IsOperator(o)` 为真。
        *   `∀N(IsComplementor(N) → o ≠ N)` 为假。因为当 `N=o` 时，`IsComplementor(o)` 为真，但 `o ≠ o` 为假，`真 → 假` 为假。
        *   合取式的第二项为假，故 `a=o` 不满足条件。
    6.  由于论域中没有任何元素 `a` 能满足 `(...)` 中的合取式，故 `∃a (...)` 在模型 **M** 中为假。

**结论:** 模型 **M** 不满足公理 A6，因此 **M** 不是扩展公理集 **T'** 的模型。公理 A6 成功地排除了此二元模型。

### **8. 互吸算子对与扩展系统的一致性**

本节引入一条新的公理 A7，它断言存在一对独特的、互为吸收的算子。随后，我们将为此最终的公理系统 **T''** 构造一个无限模型，从而证明其一致性。

#### **8.1 新定义与公理 (D6 & A7)**

**D6: 互为吸收 (ReciprocallyAbsorbing)**
`ReciprocallyAbsorbing(f, g) := P(f, g, f) ∧ P(g, f, g)`

**A7: 互吸算子对公理 (Axiom of Reciprocally Absorbing Operators)**
`∃f ∃g ( f ≠ g ∧ IsOperator(f) ∧ IsOperator(g) ∧ ReciprocallyAbsorbing(f, g) )`

设 **T''** = **T'** ∪ {A7} = {A1, A2, A3, A4, A5, A6, A7} 为最终的公理集。

#### **8.2 最终无限模型构造 ($M''_∞$)**

定义 **L**-结构 **$M''_∞$** = ⟨ D''_∞, P''_∞ᴹ ⟩ 如下：

1.  **论域 (Domain):** D''_∞ = `ℤ` (全体整数集合)。

2.  **谓词 `P` 的释义 (Interpretation):** P''_∞ᴹ ⊂ D''_∞ × D''_∞ × D''_∞ 定义为：
    P''_∞ᴹ = { (a, b, c) ∈ ℤ³ | (a = 0 ∧ c = 1 - b) ∨ (a ∈ 2ℤ \ {0} ∧ b = 0 ∧ c = 1 - a) } ∪ { (2, 4, 2), (4, 2, 4) }
    其中 `2ℤ` 代表偶数集。

#### **8.3 T'' 的可满足性证明**

我们将证明模型 **$M''_∞$** 满足公理集 **T''** 中的所有公理。

*   **证明 $M''_∞$ ⊨ A1: Uniqueness**
    `∀a ∀b ∀c₁ ∀c₂ ( (P(a, b, c₁) ∧ P(a, b, c₂)) → c₁ = c₂ )`
    1.  若 a=0，则 `P(0, b, c)` 为真要求 `c = 1-b`。对于给定的 `b`，`c` 是唯一确定的。故 `c₁ = c₂`。
    2.  若 a ∈ 2ℤ \ {0, 2, 4}，则 `P(a, b, c)` 为真要求 `b=0` 且 `c=1-a`。对于给定的 `a` 和 `b=0`，`c` 是唯一确定的。
    3.  若 a=2，`P(2, b, c)` 为真要求 `(b=0 ∧ c=-1)` 或 `(b=4 ∧ c=2)`。对于给定的 `b` (`b=0` 或 `b=4`)，`c` 是唯一确定的。对于任何其他 `b` 值，不存在 `c` 使得 `P(2,b,c)` 为真。
    4.  若 a=4，`P(4, b, c)` 为真要求 `(b=0 ∧ c=-3)` 或 `(b=2 ∧ c=4)`。对于给定的 `b` (`b=0` 或 `b=2`)，`c` 是唯一确定的。
    5.  若 `a` 为奇数，`P(a, b, c)` 恒为假，蕴含式前件为假，公式为真。
    因此，公理 A1 成立。

*   **证明 $M''_\infty \models$ A2: Extensionality**

    `∀a ∀b ( a = b ↔ ( ∀x ∀y ( (P(a, x, y) ↔ P(b, x, y)) ∧ (P(x, a, y) ↔ P(x, b, y)) ∧ (P(x, y, a) ↔ P(x, y, b)) ) ) )`

    我们将分别证明该双条件句的两个方向。

    1.  **(`→`) 方向:** `∀a ∀b ( a = b → ( ... ) )`
        此方向是等词公理 `∀x∀y (x=y → (φ(x) → φ(y)))` 的直接推论。若 `a=b`，则在右侧的全称量化公式中，`a` 的所有出现均可替换为 `b`，从而得到一个重言式。

    2.  **(`←`) 方向:** `∀a ∀b ( ( ... ) → a = b )`
        我们证明其逆否命题：`∀a ∀b ( a ≠ b → ¬( ... ) )`。
        设 `a, b ∈ ℤ` 为任意两个不同元素，即 `a ≠ b`。我们需要证明存在 `x, y ∈ ℤ`，使得右侧的合取范式为假。这等价于证明以下三个析取项之一为真：
        *   `P(a, x, y) ⇎ P(b, x, y)`
        *   `P(x, a, y) ⇎ P(x, b, y)`
        *   `P(x, y, a) ⇎ P(x, y, b)`
        （此处 `⇎` 表示 `↔` 的否定）

        我们根据 `a` 和 `b` 的奇偶性进行分类讨论。

        *   **情形 1: `a` 为偶数，`b` 为奇数。**
            令 `x = 0`，`y = 1 - a`。
            *   根据 `P''_∞ᴹ` 的定义，由于 `a` 是偶数，三元组 `(a, 0, 1-a)` 属于 `P''_∞ᴹ`。因此 `P(a, 0, 1-a)` 为真。
            *   根据 `P''_∞ᴹ` 的定义，由于 `b` 是奇数，`P(b, x, y)` 对所有 `x, y` 恒为假。因此 `P(b, 0, 1-a)` 为假。
            *   此时，`P(a, x, y) ↔ P(b, x, y)` 的真值为 `True ↔ False`，即为假。
            *   故外延性条件不满足。

        *   **情形 2: `a` 和 `b` 均为奇数。**
            令 `x = 0`，`y = 1 - a`。
            *   根据 `P''_∞ᴹ` 的定义，`P(0, a, y)` 为真当且仅当 `y = 1 - a`。因此 `P(0, a, 1-a)` 为真。
            *   根据 `P''_∞ᴹ` 的定义，`P(0, b, y)` 为真当且仅当 `y = 1 - b`。由于 `a ≠ b`，我们有 `1 - a ≠ 1 - b`。因此 `P(0, b, 1-a)` 为假。
            *   此时，`P(x, a, y) ↔ P(x, b, y)` 的真值为 `True ↔ False`，即为假。
            *   故外延性条件不满足。

        *   **情形 3: `a` 和 `b` 均为偶数。**
            令 `x = 0`，`y = 1 - a`。
            *   根据 `P''_∞ᴹ` 的定义，由于 `a` 是偶数，`P(a, 0, 1-a)` 为真。
            *   根据 `P''_∞ᴹ` 的定义，由于 `b` 是偶数，`P(b, 0, y)` 为真当且仅当 `y = 1 - b`。由于 `a ≠ b`，我们有 `1 - a ≠ 1 - b`。因此 `P(b, 0, 1-a)` 为假。（此论证不适用于 `(2,4,2)` 和 `(4,2,4)` 的特殊情况，但此处 `x=0`，不触发特殊情况）。
            *   此时，`P(a, x, y) ↔ P(b, x, y)` 的真值为 `True ↔ False`，即为假。
            *   故外延性条件不满足。

    由于以上三种情形（以及与情形1对称的 `a` 为奇数 `b` 为偶数的情形）已覆盖所有 `a ≠ b` 的可能性，我们证明了对于任何 `a ≠ b`，外延性条件都不成立。因此，其逆否命题 `( ... ) → a = b` 成立。

    综合两个方向，公理 A2 成立。

*   **证明 $M''_∞$ ⊨ A3: Restricted Conjugation**
    `∀f ( IsFunction(f) → ∃g (...) )`
    1.  在 `$M''_∞$` 中，`IsFunction(f)` 成立当且仅当 `f=0`。因为 `P(0,x,y)` 的定义 `y=1-x` 保证了对所有 `x` 存在唯一的 `y`；而对于任何 `f ≠ 0`，`P(f,x,y)` 并非对所有 `x` 都有定义，故 `¬IsFunction(f)`。
    2.  因此，我们只需验证 `f=0` 的情况。即证明 `∃g (IsFunction(g) ∧ ∀a ( IsOperator(a) → ∀b (P(0, a, b) ↔ P(a, g, b)) ) )`。
    3.  令 `g=0`。`IsFunction(0)` 为真。
    4.  需验证 `∀a ( IsOperator(a) → ∀b (P(0, a, b) ↔ P(a, 0, b)) )`。`a` 是偶数。
        *   若 `a=0`，公式为 `∀b (P(0,0,b) ↔ P(0,0,b))`，恒为真。
        *   若 `a` 为非零偶数，`P(0,a,b)` 为真 ⇔ `b = 1-a`。而 `P(a,0,b)` 为真 ⇔ `b = 1-a`。两者等价。
    因此，公理 A3 成立。

*   **证明 $M''_∞$ ⊨ A4: Existence of Complementor**
    `∃!N IsComplementor(N)`
    1.  由于 `0` 是 `$M''_∞$` 中唯一的 `Function`，故 `0` 是唯一可能满足 `IsComplementor` 的候选者。
    2.  `IsComplementor(0)` 的证明与 7.3.4 节中的证明完全相同，因为 `P(0,x,y)` 的定义未改变，且 `IsOperator` 和 `IsTerminus` 的划分（偶数/奇数）也未改变。
    因此，公理 A4 成立。

*   **证明 $M''_∞$ ⊨ A5: Complementor is Involutionary**
    `∀N (IsComplementor(N) → IsInvolution(N))`
    1.  前件 `IsComplementor(N)` 仅在 `N=0` 时为真。
    2.  后件 `IsInvolution(0)` 的证明 `∀x ∀y (P(0, x, y) → P(0, y, x))` 与 7.3.5 节中的证明完全相同。
    因此，公理 A5 成立。

*   **证明 $M''_∞$ ⊨ A6: Axiom of Inert Operators**
    `∃a ( IsOperator(a) ∧ (∀N(IsComplementor(N) → a ≠ N)) ∧ ∀t(IsTerminus(t) → ∀y(¬P(a, t, y))) )`
    1.  令 `a = 6`。
    2.  验证合取式的各部分：
        *   `IsOperator(6)`: `6` 是偶数，为真。
        *   `∀N(IsComplementor(N) → 6 ≠ N)`: 唯一的 `N` 是 `0`，`6 ≠ 0`，为真。
        *   `∀t(IsTerminus(t) → ∀y(¬P(6, t, y)))`:
            *   `t` 是 Terminus，即 `t` 是奇数。
            *   根据 P''_∞ᴹ 定义，`P(6, b, c)` 为真要求 `b=0`。
            *   由于 `t` 是奇数，所以 `t ≠ 0`。
            *   因此，`P(6, t, y)` 对任何奇数 `t` 和任何 `y` 恒为假。蕴含式成立。
    3.  存在元素 `a=6` 满足所有条件。
    因此，公理 A6 成立。

*   **证明 $M''_∞$ ⊨ A7: Axiom of Reciprocally Absorbing Operators**
    `∃f ∃g ( f ≠ g ∧ IsOperator(f) ∧ IsOperator(g) ∧ ReciprocallyAbsorbing(f, g) )`
    1.  令 `f = 2`，`g = 4`。
    2.  验证合取式的各部分：
        *   `f ≠ g`: `2 ≠ 4`，为真。
        *   `IsOperator(f) ∧ IsOperator(g)`: `2` 和 `4` 都是偶数，`IsOperator(2)` 和 `IsOperator(4)` 均为真。
        *   `ReciprocallyAbsorbing(f, g)`:
            *   `P(2, 4, 2)`: 根据 P''_∞ᴹ 的定义，`(2, 4, 2) ∈ P''_∞ᴹ`，为真。
            *   `P(4, 2, 4)`: 根据 P''_∞ᴹ 的定义，`(4, 2, 4) ∈ P''_∞ᴹ`，为真。
    3.  所有条件均满足。
    因此，公理 A7 成立。

**结论:** 模型 **$M''_∞$** 满足公理集 **T''** 中的所有公理，故公理系统 **T''** 是一致的。
