## 等价与应用理论QEA

本系统是一个基于带等词的一阶谓词逻辑的形式理论。

### **1. 形式语言 $QEA$**

**1.1. 符号表 (Alphabet)**

1.  **变量 (Variables):** 一个可数无限集 $V = \{v_0, v_1, v_2, \dots\}$。为方便起见，我们使用 $x, y, z, \dots$ 作为其元变量。
2.  **常量 (Constants):** 一个常量符号: `1`。
3.  **函数符号 (Function Symbols):**
    *   `equiv`: 二元函数符号。
    *   `·`: 二元函数符号。
4.  **谓词符号 (Predicate Symbols):**
    *   ` = `: 二元谓词符号。
5.  **逻辑联结词 (Logical Connectives):** `¬` (否定), `→` (蕴含)。
6.  **量词 (Quantifiers):** `∀` (全称量词)。
7.  **辅助符号 (Auxiliary Symbols):** `(`, `)`。

**1.2. 项 (Terms)**

$QEA$ 的项集合是满足以下条件的最小集合:
1.  任何变量 $v \in V$ 都是一个项。
2.  常量 `1` 是一个项。
3.  若 $t_1$ 和 $t_2$ 是项，则 `equiv(t₁, t₂)` 是一个项。
4.  若 $t_1$ 和 $t_2$ 是项，则 `·(t₁, t₂)` 是一个项。

**1.3. 合式公式 (Well-Formed Formulas - WFFs)**

$QEA$ 的合式公式集合是满足以下条件的最小集合:
1.  若 $t_1$ 和 $t_2$ 是项，则 `t₁ = t₂` 是一个原子公式，也是一个合式公式。
2.  若 $\phi$ 是一个合式公式，则 `(¬φ)` 是一个合式公式。
3.  若 $\phi$ 和 $\psi$ 是合式公式，则 `(φ → ψ)` 是一个合式公式。
4.  若 $\phi$ 是一个合式公式且 $x$ 是一个变量，则 `(∀x φ)` 是一个合式公式。

**1.4. 辅助定义**

为简洁起见，我们引入以下元语言缩写：
*   `(φ ∧ ψ)` 定义为 `¬(φ → ¬ψ)`
*   `(φ ∨ ψ)` 定义为 `(¬φ → ψ)`
*   `(φ ↔ ψ)` 定义为 `((φ → ψ) ∧ (ψ → φ))`
*   `(∃x φ)` 定义为 `¬(∀x ¬φ)`
*   `t₁ ≠ t₂` 定义为 `¬(t₁ = t₂)`

### **2. 公理系统 $A_{EA}$**

公理系统由逻辑公理、推理规则和非逻辑公理组成。

**2.1. 逻辑公理**

对于任意合式公式 $\phi, \psi, \chi$：

*   **A1:** $(\phi \to (\psi \to \phi))$
*   **A2:** $((\phi \to (\psi \to \chi)) \to ((\phi \to \psi) \to (\phi \to \chi)))$
*   **A3:** $((¬\phi \to ¬\psi) \to (\psi \to \phi))$
*   **A4:** $(\forall x \phi(x) \to \phi(t))$, 其中项 $t$ 在 $\phi(x)$ 中对 $x$ 是自由可代入的。
*   **A5:** $(\forall x (\phi \to \psi) \to (\phi \to \forall x \psi))$, 其中 $\phi$ 中不含自由出现的变量 $x$。

**2.2. 等词公理 (Axioms of Equality)**

对于任意变量 $x, y$ 和项 $t_1, \dots, t_n, s_1, \dots, s_n$：

*   **A6 (自反性):** $\forall x (x = x)$
*   **A7 (替换公理模式):**
    *   对于任意 $n$ 元函数符号 $f$ (在 $QEA$ 中为 `equiv` 和 `·`):
        $\forall x_1 \dots \forall x_n \forall y_1 \dots \forall y_n ( (x_1=y_1 \land \dots \land x_n=y_n) \to f(x_1, \dots, x_n) = f(y_1, \dots, y_n) )$
    *   对于任意 $n$ 元谓词符号 $P$ (在 $QEA$ 中为 ` = `)
        $\forall x_1 \dots \forall x_n \forall y_1 \dots \forall y_n ( (x_1=y_1 \land \dots \land x_n=y_n) \to (P(x_1, \dots, x_n) \to P(y_1, \dots, y_n)) )$

**2.3. 推理规则 (Rules of Inference)**

1.  **分离规则 (Modus Ponens):** 从 $\phi$ 和 $(\phi \to \psi)$，可推出 $\psi$。
2.  **全称概括 (Generalization):** 从 $\phi$，可推出 $(\forall x \phi)$。

**2.4. 非逻辑公理 (Non-Logical Axioms)**

以下公理定义了理论 $QEA$ 的特定性质。

*   **EA1 (等价定义):**
    $\forall x \forall y ( \text{equiv}(x, y) = 1 \leftrightarrow x = y )$

*   **EA2 (混合结合律):**
    $\forall x \forall y \forall z ( \text{equiv}(\cdot(x, y), z) = \cdot(x, \text{equiv}(y, z)) )$

*   **EA3 (单位元`1`的右幺性):**
    $\forall x ( \text{equiv}(x, 1) = x )$

*   **EA4 (逆性公理):**
    $∀p ( (∃x ∃y (p = \text{equiv}(x, y))) → (∃q (·(p, q) = 1 ∧ ·(q, p) = 1)) )$

### **3. 模型与一致性证明**

为了证明理论`QEA`（其非逻辑公理集为 `{EA1, EA2, EA3, EA4}`）是一致的，我们构造一个模型 `M`，并证明所有公理在该模型下为真。

#### **3.1. 模型的构造**

我们定义一个模型 `M = <U, I>`，其中 `U` 是论域，`I` 是解释函数。

1.  **论域 (Universe) U:**
    `U` 是作用于集合 `{1, 2, 3}` 上的对称群 $S_3$ 的元素集合。
    `U = {e, (12), (13), (23), (123), (132)}`，其中 `e` 是恒等置换。

2.  **解释函数 (Interpretation Function) I:**
    *   **常量解释:**
        `I(1) := e` (群 $S_3$ 的单位元)

    *   **函数符号解释:**
        对于任意 `a, b ∈ U`:
        *   `I(·)` 是一个二元函数 `·_M: U × U → U`，定义为群 $S_3$ 中的置换复合运算 `∘`。
            `·_M(a, b) := a ∘ b`
        *   `I(equiv)` 是一个二元函数 `equiv_M: U × U → U`，定义为：
            `equiv_M(a, b) := a ∘ b⁻¹` (其中 `b⁻¹` 是 `b` 在群 $S_3$ 中的逆元)

    *   **谓词符号解释:**
        `I(=)` 是 `U` 上的恒等关系。

#### **3.2. 公理验证**

我们证明模型 `M` 满足理论 `QEA` 的所有非逻辑公理。

**A. 验证公理 EA1: `∀x ∀y ( equiv(x, y) = 1 ↔ x = y )`**

在模型 `M` 中，该公理被解释为：
`∀a, b ∈ U ( equiv_M(a, b) = I(1) ↔ a = b )`
代入 `I` 的定义，即：
`∀a, b ∈ U ( a ∘ b⁻¹ = e ↔ a = b )`

**证明:**
1.  **方向 (→):** 假设 `a ∘ b⁻¹ = e`。根据群公理，将等式右侧与 `b` 复合，得到 `(a ∘ b⁻¹) ∘ b = e ∘ b`。根据结合律，`a ∘ (b⁻¹ ∘ b) = e ∘ b`。根据逆元定义，`a ∘ e = b`。根据单位元定义，`a = b`。
2.  **方向 (←):** 假设 `a = b`。根据等词公理，将等式右侧与 `b⁻¹` 复合，得到 `a ∘ b⁻¹ = b ∘ b⁻¹`。根据逆元定义，`b ∘ b⁻¹ = e`。因此，`a ∘ b⁻¹ = e`。
**结论：公理 EA1 在模型 M 中成立。**

**B. 验证公理 EA2: `∀x ∀y ∀z ( equiv(·(x, y), z) = ·(x, equiv(y, z)) )`**

在模型 `M` 中，该公理被解释为：
`∀a, b, c ∈ U ( equiv_M(·_M(a, b), c) = ·_M(a, equiv_M(b, c)) )`
代入 `I` 的定义，即：
`∀a, b, c ∈ U ( (a ∘ b) ∘ c⁻¹ = a ∘ (b ∘ c⁻¹) )`

**证明:**
该等式是群运算 `∘` 满足结合律的直接推论。
**结论：公理 EA2 在模型 M 中成立。**

**C. 验证公理 EA3: `∀x ( equiv(x, 1) = x )`**

在模型 `M` 中，该公理被解释为：
`∀a ∈ U ( equiv_M(a, I(1)) = a )`
代入 `I` 的定义，即：
`∀a ∈ U ( a ∘ e⁻¹ = a )`

**证明:**
根据单位元的定义，`e⁻¹ = e`。因此，等式左侧为 `a ∘ e`。根据单位元定义，`a ∘ e = a`。
**结论：公理 EA3 在模型 M 中成立。**

**D. 验证公理 EA4: `∀p ( (∃x ∃y (p = equiv(x, y))) → (∃q (·(p, q) = 1 ∧ ·(q, p) = 1)) )`**

在模型 `M` 中，该公理被解释为：
`∀p ∈ U ( (∃a, b ∈ U (p = equiv_M(a, b))) → (∃q ∈ U (·_M(p, q) = I(1) ∧ ·_M(q, p) = I(1))) )`
代入 `I` 的定义，即：
`∀p ∈ U ( (∃a, b ∈ U (p = a ∘ b⁻¹)) → (∃q ∈ U (p ∘ q = e ∧ q ∘ p = e)) )`

**证明:**
我们对任意 `p ∈ U` 验证此蕴含式。
1.  **分析前件 `∃a, b ∈ U (p = a ∘ b⁻¹)`:**
    对于任意给定的 `p ∈ U`，我们必须证明存在 `a, b ∈ U` 使得 `p = a ∘ b⁻¹`。令 `a := p`，`b := e`。由于 `p` 和 `e` 都是 `U` 的成员，所以 `a, b ∈ U`。计算 `a ∘ b⁻¹ = p ∘ e⁻¹ = p ∘ e = p`。因此，前件对于 `U` 中的任意元素 `p` 均成立。

2.  **分析后件 `∃q ∈ U (p ∘ q = e ∧ q ∘ p = e)`:**
    由于前件对所有 `p ∈ U` 为真，蕴含式为真当且仅当后件对所有 `p ∈ U` 也为真。后件断言：对于 `U` 中的任意元素 `p`，存在一个元素 `q ∈ U`，它是 `p` 的双边逆元。
    根据群的定义，`U` (即群 $S_3$) 中的每一个元素 `p` 都存在一个唯一的逆元 `p⁻¹`，且 `p⁻¹ ∈ U`。令 `q := p⁻¹`，则 `p ∘ q = p ∘ p⁻¹ = e` 且 `q ∘ p = p⁻¹ ∘ p = e`。因此，后件对于 `U` 中的任意元素 `p` 均成立。

**结论：公理 EA4 在模型 M 中成立。**

**总结**
由于理论 `QEA` 的所有非逻辑公理都在模型 `M` 中得到满足，`M` 是 `QEA` 的一个模型。根据哥德尔完全性定理，一个理论存在模型是其相容（一致）的充分必要条件。因此，理论 `QEA` 是一致的。此外，由于模型 `M` 的基数（6）大于2，且其代数结构为非阿贝尔群，故该模型是一个非平凡模型。
