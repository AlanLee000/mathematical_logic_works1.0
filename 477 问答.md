### **逻辑系统 LEI (Logical System LEI)**

#### **1. 形式语言 $\mathcal{L}_{ei}$**

**1.1 字母表 (Alphabet)**

*   **常量 (Constants):**
    $Q, A$
*   **一元算子 (Unary Operator):**
    $?(\cdot)$
*   **二元算子 (Binary Operators):**
    $(\cdot \text{ eval } \cdot)$, $[\cdot, \cdot]$
*   **辅助符号 (Auxiliary Symbols):**
    $(, ), ,$

**1.2 合式公式 (Well-formed Formulas, WFFs)**

合式公式的集合通过以下规则归纳定义：

1.  **基础 (Base):** $Q$ 和 $A$ 是合式公式。
2.  **归纳 (Induction):**
    a. 若 $\phi$ 是一个合式公式，则 $?(\phi)$ 也是一个合式公式。
    b. 若 $\phi$ 和 $\psi$ 是合式公式，则 $(\phi \text{ eval } \psi)$ 也是一个合式公式。
    c. 若 $\phi$ 和 $\psi$ 是合式公式，则 $[\phi, \psi]$ 也是一个合式公式。
3.  **封闭 (Closure):** 仅由有限次应用上述规则生成的表达式是合式公式。

**1.3 子公式、语境与替换**

**定义 1.3.1 (子公式)**

一个公式 $\phi$ 的**子公式**集合，记为 $Sub(\phi)$，归纳定义如下：
1.  若 $\phi$ 是常量 $Q$ 或 $A$，则 $Sub(\phi) = \{\phi\}$。
2.  若 $\phi$ 的形式为 $?(\alpha)$，则 $Sub(\phi) = \{?(\alpha)\} \cup Sub(\alpha)$。
3.  若 $\phi$ 的形式为 $(\alpha \text{ eval } \beta)$，则 $Sub(\phi) = \{(\alpha \text{ eval } \beta)\} \cup Sub(\alpha) \cup Sub(\beta)$。
4.  若 $\phi$ 的形式为 $[\alpha, \beta]$，则 $Sub(\phi) = \{[\alpha, \beta]\} \cup Sub(\alpha) \cup Sub(\beta)$。

**定义 1.3.2 (语境与替换)**

1.  一个**语境 (Context)** 是一个带有一个“空位” $(\cdot)$ 的表达式，将任意合式公式填入该空位即可得到一个合式公式。我们用 $C[\cdot]$ 表示一个语境。
2.  给定一个语境 $C[\cdot]$ 和一个公式 $\phi$，我们用 $C[\phi]$ 表示将 $\phi$ 填入 $C[\cdot]$ 的空位后得到的合式公式。此时，$\phi$ 是 $C[\phi]$ 的一个特定的子公式出现 (a designated occurrence of the subformula)。
3.  **替换 (Substitution)** 是指在一个语境下，用一个公式替换另一个公式。例如，从 $C[\phi]$ 变为 $C[\psi]$ 就是将子公式 $\phi$ 的特定出现替换为 $\psi$。

#### **2. 公理系统 $\mathcal{S}_{ei}$**

**2.1 公理模式 (Axiom Schemata)**

对于任意的合式公式 $\phi, \psi$：

*   **A1:**
    $$ ([A, ?(\phi)] \text{ eval } \phi) $$
*   **A2:**
    $$ ([Q, (\phi \text{ eval } \psi)] \text{ eval } ?(\psi)) $$
*   **A3:**
    $$ ([Q, \phi] \text{ eval } ?(\phi)) $$

**2.2 推理规则 (Inference Rules)**

令 $\vdash \phi$ 表示“$\phi$ 是系统中的一个定理”。

*   **R1 (自反性):** 对于任意合式公式 $\phi$，可以引入 $(\phi \text{ eval } \phi)$。
    $$ \frac{}{(\phi \text{ eval } \phi)} $$


*   **R2 (替换规则 / Rule of Substitution):**

    若 $(\phi \text{ eval } \psi)$ 是一个定理，且 $C[\phi]$ 是一个定理（其中 $C[\cdot]$ 是任意语境），那么 $C[\psi]$ 也是一个定理。

    $$ \frac{\vdash (\phi \text{ eval } \psi) \quad \vdash C[\phi]}{\vdash C[\psi]} $$

### **3. 模型论语义 (Model-Theoretic Semantics)**

本节为形式语言 $\mathcal{L}_{ei}$ 及公理系统 $\mathcal{S}_{ei}$ 提供一个模型论语义。

#### **3.1 LEI-结构 (LEI-Structure)**

**定义 3.1.1** 一个 **LEI-结构** (或模型) $\mathcal{M}$ 是一个三元组 $\mathcal{M} = (D, c_Q, c_A)$，其中：
1.  $D$ 是一个非空集合，称为 **数据域 (Domain of Data)**。
2.  $c_Q$ 和 $c_A$ 是两个特殊对象，分别称为 **查询语境 (Query Context)** 和 **回答语境 (Answer Context)**。
3.  $D$, $\{c_Q\}$, 和 $\{c_A\}$ 三个集合两两不相交。

#### **3.2 语义域 (Semantic Universe)**

**定义 3.2.1** 对于给定的 LEI-结构 $\mathcal{M}$，其 **语义域** $U_{\mathcal{M}}$ 是满足以下条件的最小集合：
1.  $D \subseteq U_{\mathcal{M}}$。
2.  $\{c_Q, c_A\} \subseteq U_{\mathcal{M}}$。
3.  若 $u \in U_{\mathcal{M}}$，则有序对 $\langle \text{query}, u \rangle \in U_{\mathcal{M}}$。

我们将形如 $\langle \text{query}, u \rangle$ 的元素称为 **查询对象 (Query Objects)**。

#### **3.3 求值函数 (Valuation Function)**

**定义 3.3.1** 对于一个 LEI-结构 $\mathcal{M}$，**求值函数** $v_{\mathcal{M}}: \text{WFFs} \text{ eval } U_{\mathcal{M}}$ 是一个将所有合式公式映射到语义域的函数，其定义如下：

1.  **基础 (Base Cases):**
    *   $v_{\mathcal{M}}(Q) = c_Q$
    *   $v_{\mathcal{M}}(A) = c_A$

2.  **归纳 (Inductive Cases):** 对于任意合式公式 $\phi, \psi$：
    *   $v_{\mathcal{M}}(?(\phi)) = \langle \text{query}, v_{\mathcal{M}}(\phi) \rangle$
    *   $v_{\mathcal{M}}((\phi \text{ eval } \psi)) = v_{\mathcal{M}}(\psi)$
    *   $v_{\mathcal{M}}([\phi, \psi])$ 的值由以下互斥条件确定：
        a.  若 $v_{\mathcal{M}}(\phi) = c_A$ 且 $v_{\mathcal{M}}(\psi) = \langle \text{query}, u \rangle$ 对于某个 $u \in U_{\mathcal{M}}$ 成立，则 $v_{\mathcal{M}}([\phi, \psi]) = u$。
        b.  若 $v_{\mathcal{M}}(\phi) = c_Q$，则 $v_{\mathcal{M}}([\phi, \psi]) = \langle \text{query}, v_{\mathcal{M}}(\psi) \rangle$。
        c.  若以上条件均不满足，则 $v_{\mathcal{M}}([\phi, \psi]) = v_{\mathcal{M}}(\psi)$。

#### **3.4 有效性 (Validity)**

**定义 3.4.1**
1.  一个公式 $(\phi \text{ eval } \psi)$ 在模型 $\mathcal{M}$ 中是 **有效的 (valid)**，记为 $\mathcal{M} \models (\phi \text{ eval } \psi)$，当且仅当 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$。
2.  一个公式 $(\phi \text{ eval } \psi)$ 是 **普遍有效的 (universally valid)**，记为 $\models (\phi \text{ eval } \psi)$，当且仅当对于任意 LEI-结构 $\mathcal{M}$，都有 $\mathcal{M} \models (\phi \text{ eval } \psi)$。

#### **3.5 可靠性 (Soundness)**

**定理 3.5.1 (可靠性定理)**
如果一个公式 $\phi$ 是系统 $\mathcal{S}_{ei}$ 的一个定理 (即 $\vdash \phi$)，并且 $\phi$ 的形式为 $(\psi \text{ eval } \chi)$，那么该公式是普遍有效的 (即 $\models (\psi \text{ eval } \chi)$)。

**证明:**
证明通过对 $\mathcal{S}_{ei}$ 中定理的派生长度进行归纳。我们只需证明所有公理都是普遍有效的，并且所有推理规则都保持普遍有效性。

**引理 3.5.2 (公理的有效性)**
公理模式 A1, A2, A3 均为普遍有效的。

**证明:** 令 $\mathcal{M}$ 为任意 LEI-结构。

*   **对于 A1: $([A, ?(φ)] \text{ eval } φ)$**
    我们需要证明 $\models ([A, ?(\phi)] \text{ eval } \phi)$，即对于任意 $\phi$，$v_{\mathcal{M}}([A, ?(\phi)]) = v_{\mathcal{M}}(\phi)$。
    1.  令 $v_{\mathcal{M}}(\phi) = u$。根据定义 3.3.1，$v_{\mathcal{M}}(?(\phi)) = \langle \text{query}, u \rangle$。
    2.  求值左侧 $v_{\mathcal{M}}([A, ?(\phi)])$：
        *   $v_{\mathcal{M}}(A) = c_A$。
        *   $v_{\mathcal{M}}(?(\phi)) = \langle \text{query}, u \rangle$。
        *   这满足定义 3.3.1 中求值 $[\cdot, \cdot]$ 的条件 (a)。
        *   因此，$v_{\mathcal{M}}([A, ?(\phi)]) = u$。
    3.  求值右侧 $v_{\mathcal{M}}(\phi) = u$。
    4.  左右两侧相等，故 A1 是普遍有效的。

*   **对于 A2: $([Q, (φ \text{ eval } ψ)] \text{ eval } ?(ψ))$**
    我们需要证明 $\models ([Q, (\phi \text{ eval } \psi)] \text{ eval } ?(\psi))$，即 $v_{\mathcal{M}}([Q, (\phi \text{ eval } \psi)]) = v_{\mathcal{M}}(?(\psi))$。
    1.  求值左侧 $v_{\mathcal{M}}([Q, (\phi \text{ eval } \psi)])$：
        *   $v_{\mathcal{M}}(Q) = c_Q$。
        *   这满足定义 3.3.1 中求值 $[\cdot, \cdot]$ 的条件 (b)。
        *   因此，$v_{\mathcal{M}}([Q, (\phi \text{ eval } \psi)]) = \langle \text{query}, v_{\mathcal{M}}((\phi \text{ eval } \psi)) \rangle$。
        *   根据定义，$v_{\mathcal{M}}((\phi \text{ eval } \psi)) = v_{\mathcal{M}}(\psi)$。
        *   所以，$v_{\mathcal{M}}([Q, (\phi \text{ eval } \psi)]) = \langle \text{query}, v_{\mathcal{M}}(\psi) \rangle$。
    2.  求值右侧 $v_{\mathcal{M}}(?(\psi)) = \langle \text{query}, v_{\mathcal{M}}(\psi) \rangle$。
    3.  左右两侧相等，故 A2 是普遍有效的。

*   **对于 A3: $([Q, φ] \text{ eval } ?(φ))$**
    我们需要证明 $\models ([Q, \phi] \text{ eval } ?(\phi))$，即 $v_{\mathcal{M}}([Q, \phi]) = v_{\mathcal{M}}(?(\phi))$。
    1.  求值左侧 $v_{\mathcal{M}}([Q, \phi])$：
        *   $v_{\mathcal{M}}(Q) = c_Q$。
        *   这满足定义 3.3.1 中求值 $[\cdot, \cdot]$ 的条件 (b)。
        *   因此，$v_{\mathcal{M}}([Q, \phi]) = \langle \text{query}, v_{\mathcal{M}}(\phi) \rangle$。
    2.  求值右侧 $v_{\mathcal{M}}(?(\phi)) = \langle \text{query}, v_{\mathcal{M}}(\phi) \rangle$。
    3.  左右两侧相等，故 A3 是普遍有效的。

**引理 3.5.3 (推理规则的保效性)**
推理规则 R1 和 R2 保持普遍有效性。

**证明:**

*   **对于 R1 (自反性):**
    该规则允许引入 $(\phi \text{ eval } \phi)$。我们需要证明 $\models (\phi \text{ eval } \phi)$。
    根据定义 3.4.1，这等价于证明对于任意 $\mathcal{M}$ 和 $\phi$，$v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\phi)$。
    此等式根据等号的自反性显然成立。

*   **对于 R2 (替换规则 / Rule of Substitution):**

该规则陈述为：若 $\vdash (\phi \text{ eval } \psi)$ 且 $\vdash C[\phi]$，则 $\vdash C[\psi]$。
为证明此规则保持普遍有效性，我们首先引入并证明一个关于语义求值函数的组合性引理。

**引理 3.5.3.1 (组合性引理 / Compositionality Lemma)**
对于任意 LEI-结构 $\mathcal{M}$，任意语境 $C[\cdot]$，以及任意合式公式 $\phi, \psi$：
若 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$，则 $v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(C[\psi])$。

**证明：**
通过对语境 $C[\cdot]$ 的结构进行归纳来证明。

1.  **基础情况 (Base Case):**
    $C[\cdot]$ 是空位 $(\cdot)$。
    此时 $C[\phi] = \phi$ 且 $C[\psi] = \psi$。
    引理的陈述变为：若 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$，则 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$。
    此为恒等命题，显然成立。

2.  **归纳步骤 (Inductive Step):**
    假设对于语境 $C'[\cdot]$，引理成立（此为归纳假设，IH）。即，若 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$，则 $v_{\mathcal{M}}(C'[\phi]) = v_{\mathcal{M}}(C'[\psi])$。
    我们需要证明对于由 $C'[\cdot]$ 构造的更复杂的语境，引理依然成立。令 $\alpha$ 为任意合式公式。

    *   **情况 a:** $C[\cdot] = ?(C'[\cdot])$
        我们有 $v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(?(C'[\phi])) = \langle \text{query}, v_{\mathcal{M}}(C'[\phi]) \rangle$。
        同时 $v_{\mathcal{M}}(C[\psi]) = v_{\mathcal{M}}(?(C'[\psi])) = \langle \text{query}, v_{\mathcal{M}}(C'[\psi]) \rangle$。
        根据归纳假设，$v_{\mathcal{M}}(C'[\phi]) = v_{\mathcal{M}}(C'[\psi])$。因此，$\langle \text{query}, v_{\mathcal{M}}(C'[\phi]) \rangle = \langle \text{query}, v_{\mathcal{M}}(C'[\psi]) \rangle$，即 $v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(C[\psi])$。

    *   **情况 b:** $C[\cdot] = (C'[\cdot] \text{ eval } \alpha)$
        根据求值函数定义 3.3.1，$v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}((C'[\phi] \text{ eval } \alpha)) = v_{\mathcal{M}}(\alpha)$。
        同理，$v_{\mathcal{M}}(C[\psi]) = v_{\mathcal{M}}((C'[\psi] \text{ eval } \alpha)) = v_{\mathcal{M}}(\alpha)$。
        因此，$v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(C[\psi])$。

    *   **情况 c:** $C[\cdot] = (\alpha \text{ eval } C'[\cdot])$
        根据求值函数定义 3.3.1，$v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}((\alpha \text{ eval } C'[\phi])) = v_{\mathcal{M}}(C'[\phi])$。
        同理，$v_{\mathcal{M}}(C[\psi]) = v_{\mathcal{M}}((\alpha \text{ eval } C'[\psi])) = v_{\mathcal{M}}(C'[\psi])$。
        根据归纳假设，$v_{\mathcal{M}}(C'[\phi]) = v_{\mathcal{M}}(C'[\psi])$，因此 $v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(C[\psi])$。

    *   **情况 d:** $C[\cdot] = [C'[\cdot], \alpha]$
        令 $u_1 = v_{\mathcal{M}}(C'[\phi])$ 且 $u_2 = v_{\mathcal{M}}(C'[\psi])$。根据归纳假设，$u_1 = u_2$。
        $v_{\mathcal{M}}(C[\phi])$ 的值由 $u_1$ 和 $v_{\mathcal{M}}(\alpha)$ 根据定义 3.3.1 的三种情况确定。
        $v_{\mathcal{M}}(C[\psi])$ 的值由 $u_2$ 和 $v_{\mathcal{M}}(\alpha)$ 根据相同的定义确定。
        由于 $u_1 = u_2$，两个求值过程的输入完全相同，因此其结果必然相同，即 $v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(C[\psi])$。

    *   **情况 e:** $C[\cdot] = [\alpha, C'[\cdot]]$
        令 $w_1 = v_{\mathcal{M}}(C'[\phi])$ 且 $w_2 = v_{\mathcal{M}}(C'[\psi])$。根据归纳假设，$w_1 = w_2$。
        $v_{\mathcal{M}}(C[\phi])$ 的值由 $v_{\mathcal{M}}(\alpha)$ 和 $w_1$ 根据定义 3.3.1 的三种情况确定。
        $v_{\mathcal{M}}(C[\psi])$ 的值由 $v_{\mathcal{M}}(\alpha)$ 和 $w_2$ 根据相同的定义确定。
        由于 $w_1 = w_2$，两个求值过程的输入完全相同，因此其结果必然相同，即 $v_{\mathcal{M}}(C[\phi]) = v_{\mathcal{M}}(C[\psi])$。

通过结构归纳法，组合性引理得证。

**R2 保效性主证明:**
我们需要证明，如果前提 $\models (\phi \text{ eval } \psi)$ 和 $\models C[\phi]$ 成立，那么结论 $\models C[\psi]$ 也成立。

1.  **归纳假设:**
    a. $\models (\phi \text{ eval } \psi)$。根据有效性定义 3.4.1，这意味着对于任意 LEI-结构 $\mathcal{M}$，都有 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$。
    b. $\models C[\phi]$。由于 $\mathcal{S}_{ei}$ 系统中的所有公理和 R1 的结论都是 $(\text{ eval })$ 形式的公式，并且 R2 规则的结论也必然是 $(\text{ eval })$ 形式，我们可以通过对证明长度的简单归纳得出，系统中的所有定理都具有 $(\alpha \text{ eval } \beta)$ 的形式。因此，我们可以将 $C[\phi]$ 写为 $(\alpha \text{ eval } \beta)$ 的形式。那么，假设 $\models (\alpha \text{ eval } \beta)$ 意味着对于任意 LEI-结构 $\mathcal{M}$，都有 $v_{\mathcal{M}}(\alpha) = v_{\mathcal{M}}(\beta)$。

2.  **待证目标:**
    证明 $\models C[\psi]$。其中 $C[\psi]$ 是通过将 $C[\phi]$ 中 $\phi$ 的一个特定出现替换为 $\psi$ 而得到的公式。

3.  **证明过程:**
    令 $\mathcal{M}$ 为一个任意的 LEI-结构。
    $C[\phi]$ 的形式为 $(\alpha \text{ eval } \beta)$。被替换的 $\phi$ 的特定出现必然位于 $\alpha$ 或 $\beta$ 之中。

    *   **情况 1:** $\phi$ 的特定出现位于 $\alpha$ 中。
        此时，我们可以将 $\alpha$ 表示为一个包含 $\phi$ 的语境，记为 $\alpha = D[\phi]$，其中 $D[\cdot]$ 是一个子语境。
        那么 $C[\phi]$ 就是 $(D[\phi] \text{ eval } \beta)$，而 $C[\psi]$ 就是 $(D[\psi] \text{ eval } \beta)$。
        根据假设 (b)，我们有 $v_{\mathcal{M}}(D[\phi]) = v_{\mathcal{M}}(\beta)$。
        根据假设 (a)，我们有 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$。
        将组合性引理 3.5.3.1 应用于语境 $D[\cdot]$，我们从 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$ 推导出 $v_{\mathcal{M}}(D[\phi]) = v_{\mathcal{M}}(D[\psi])$。
        根据等号的传递性，由 $v_{\mathcal{M}}(D[\psi]) = v_{\mathcal{M}}(D[\phi])$ 和 $v_{\mathcal{M}}(D[\phi]) = v_{\mathcal{M}}(\beta)$，可得 $v_{\mathcal{M}}(D[\psi]) = v_{\mathcal{M}}(\beta)$。
        根据有效性定义 3.4.1，这意味着 $\mathcal{M} \models (D[\psi] \text{ eval } \beta)$，即 $\mathcal{M} \models C[\psi]$。

    *   **情况 2:** $\phi$ 的特定出现位于 $\beta$ 中。
        此时，我们可以将 $\beta$ 表示为 $\beta = D[\phi]$。
        那么 $C[\phi]$ 就是 $(\alpha \text{ eval } D[\phi])$，而 $C[\psi]$ 就是 $(\alpha \text{ eval } D[\psi])$。
        根据假设 (b)，我们有 $v_{\mathcal{M}}(\alpha) = v_{\mathcal{M}}(D[\phi])$。
        根据假设 (a)，我们有 $v_{\mathcal{M}}(\phi) = v_{\mathcal{M}}(\psi)$。
        同样应用组合性引理，我们推导出 $v_{\mathcal{M}}(D[\phi]) = v_{\mathcal{M}}(D[\psi])$。
        根据等号的传递性，由 $v_{\mathcal{M}}(\alpha) = v_{\mathcal{M}}(D[\phi])$ 和 $v_{\mathcal{M}}(D[\phi]) = v_{\mathcal{M}}(D[\psi])$，可得 $v_{\mathcal{M}}(\alpha) = v_{\mathcal{M}}(D[\psi])$。
        根据有效性定义 3.4.1，这意味着 $\mathcal{M} \models (\alpha \text{ eval } D[\psi])$，即 $\mathcal{M} \models C[\psi]$。

由于 $\mathcal{M}$ 是任意选择的，因此在两种情况下，我们都证明了对于所有 LEI-结构，$\mathcal{M} \models C[\psi]$ 均成立。
所以，$\models C[\psi]$。
推理规则 R2 保持普遍有效性。

公理和推理规则均满足可靠性要求，定理 3.5.1 证毕。

#### **3.6 非平凡性与非退化性 (Non-triviality and Non-degeneracy)**

**命题 3.6.1 (非平凡性)**
存在一个 LEI-结构 $\mathcal{M}$ 和两个公式 $\phi, \psi$，使得 $\mathcal{M} \not\models (\phi \text{ eval } \psi)$。

**证明:**
构造一个 LEI-结构 $\mathcal{M}$，其中数据域 $D = \{d_1, d_2\}$ 且 $d_1 \ne d_2$。
考虑两个常量 $Q, A$ 作为公式。
$v_{\mathcal{M}}(Q) = c_Q$
$v_{\mathcal{M}}(A) = c_A$
根据定义 3.1.1，$c_Q \ne c_A$。因此 $v_{\mathcal{M}}(Q) \ne v_{\mathcal{M}}(A)$。
根据定义 3.4.1，$\mathcal{M} \not\models (Q \text{ eval } A)$。
因此，该语义学是 **非平凡的**，即并非所有 $(\text{ eval })$ 形式的公式都有效。

**命题 3.6.2 (非退化性)**
对于任意 LEI-结构 $\mathcal{M}$，只要 $|D| \ge 1$，那么语义域 $U_{\mathcal{M}}$ 的三个核心子集 $D$, $\{c_Q, c_A\}$ 以及查询对象集合 $\{ \langle \text{query}, u \rangle \mid u \in U_{\mathcal{M}} \}$ 都是两两不相交的。

**证明:**
1.  $D \cap \{c_Q, c_A\} = \emptyset$ 直接由定义 3.1.1 保证。
2.  $D$ 中的元素是基础数据，而查询对象是形如 $\langle \text{query}, u \rangle$ 的有序对。假设 $d \in D$ 且 $d = \langle \text{query}, u \rangle$。这在标准集合论中是不可能的，除非 $D$ 的元素本身被构造成有序对，但我们可以选择一个 $D$ 来避免这种情况 (例如，令 $D$ 为自然数集)。因此 $D$ 与查询对象集合不相交。
3.  $\{c_Q, c_A\}$ 中的元素是原子符号，而查询对象是有序对。同理，它们是不相交的。
因此，该语义学是 **非退化的**，即不同概念类型的实体在语义上保持区分，不会坍缩为同一类型。
