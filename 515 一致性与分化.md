### 形式系统LCD

#### 1. 句法定义 (Syntax)

**字母表：**
*   **变量：** $A, B, C, \dots$
*   **一元算子：** $\neg$
*   **二元算子：** $\text{Con}, \text{Ext}$
*   **辅助符号：** $(, )$

**合式公式 (WFF) 定义：**
1.  若 $A$ 是变量，则 $A$ 是合式公式。
2.  若 $A$ 是合式公式，则 $\neg(A)$ 是合式公式。
3.  若 $A, B$ 是合式公式，则 $(A \text{ Con } B)$ 与 $(A \text{ Ext } B)$ 是合式公式。

---

#### 2. 公理模式 (Axiom Schemata)

$$ \overline{\neg(A \text{ Con } A)} \quad \text{(A1. 反自反律)} $$

$$ \overline{(A \text{ Con } A) \text{ Con } \neg(A \text{ Con } A)} \quad \text{(A2. 真值阶序律)} $$

---

#### 3. 推导规则 (Inference Rules)

**I. 基础结构规则**

$$ \frac{A \text{ Con } B, \quad B \text{ Con } C}{A \text{ Con } C} \quad \text{(R1.1 传递律)} $$

$$ \frac{A \text{ Con } B}{\neg(B \text{ Con } A)} \quad \text{(R1.2 非对称律)} $$

$$ \frac{\neg(\neg(A))}{A} \quad \text{(R1.3 双重否定消除)} $$

**II. 一致性迭代规则**

$$ \frac{A \text{ Con } B}{A \text{ Con } (A \text{ Con } B)} \quad \text{(R2.1 正向迭代)} $$

$$ \frac{A \text{ Con } (A \text{ Con } B)}{A \text{ Con } B} \quad \text{(R2.2 逆向塌缩)} $$

**III. 分化与排斥规则**

$$ \frac{A \text{ Ext } B}{\neg(A \text{ Con } B)} \quad \text{(R3.1 分化阻断)} $$

$$ \frac{A \text{ Con } \neg(B)}{\neg(A \text{ Ext } B)} \quad \text{(R3.2 判定的确定性)} $$

$$ \frac{A \text{ Ext } B}{B \text{ Ext } A} \quad \text{(R3.3 分化对称性)} $$

$$ \frac{\neg(A \text{ Con } \neg(A)), \quad \neg(\neg(A) \text{ Con } A)}{A \text{ Ext } \neg(A)} \quad \text{(R3.4 分化显现律)} $$

**IV. 混合算子扩张规则**

$$ \frac{A \text{ Con } (B \text{ Ext } C)}{A \text{ Con } (B \text{ Ext } \neg(C))} \quad \text{(R4.1 分化的否定无关性)} $$

$$ \frac{A \text{ Con } (B \text{ Con } C), \quad A \text{ Ext } D}{A \text{ Con } (D \text{ Con } (B \text{ Con } C))} \quad \text{(R4.2 独立前件引入)} $$

$$ \frac{(A \text{ Con } B) \text{ Ext } C}{A \text{ Con } (C \text{ Con } B)} \quad \text{(R4.3 元分化转换 I)} $$

$$ \frac{(A \text{ Con } B) \text{ Ext } C}{\neg(A \text{ Con } (\neg(C) \text{ Con } B))} \quad \text{(R4.4 元分化转换 II)} $$

$$ \frac{\neg(A \text{ Con } B), \quad B \text{ Ext } C}{\neg(A \text{ Con } (C \text{ Con } B))} \quad \text{(R4.5 弱链阻断)} $$


#### 4. 推演 (Derivation)
如果存在一个有限的公式序列 $\alpha_1, \dots, \alpha_n$，使得每一个 $\alpha_i$ 要么是公理，要么是由序列中先前的公式通过推理规则得到，则称 $\alpha_n$ 是系统 $\mathcal{S}$ 的定理，记作 $\vdash_\mathcal{S} \alpha_n$。


#### 5. 退化语义模型 (Semantics)

为了确立系统 LCD 的相容性与非平凡性，我们构造一个基于**有限秩布尔代数**的模型结构 $\mathfrak{M}$。

##### 5.1 模型结构定义
模型 $\mathfrak{M}$ 定义为一个四元组 $\mathfrak{M} = \langle \mathcal{B}, \neg^\mathfrak{M}, \rho, V \rangle$，其中：

1.  **基础域 $\mathcal{B}$**：是一个包含四个元素的布尔格 $\mathcal{B} = \{ \bot, \alpha, \beta, \top \}$。
    *   $\bot$：格的最小元（逻辑假）。
    *   $\top$：格的最大元（逻辑真）。
    *   $\alpha, \beta$：互补的原子元，满足 $\alpha \land \beta = \bot, \alpha \lor \beta = \top$。

2.  **一元算子解释 $\neg^\mathfrak{M}$**：$\mathcal{B}$ 上的标准布尔补运算。
    *   $\neg^\mathfrak{M} \top = \bot, \quad \neg^\mathfrak{M} \bot = \top$
    *   $\neg^\mathfrak{M} \alpha = \beta, \quad \neg^\mathfrak{M} \beta = \alpha$

3.  **秩函数 $\rho$**：映射 $\rho: \mathcal{B} \to \{1, 2, 3\}$，定义逻辑强度如下：
    *   $\rho(\bot) = 3$
    *   $\rho(\alpha) = 2$
    *   $\rho(\beta) = 2$
    *   $\rho(\top) = 1$

4.  **赋值函数 $V$**：将公式映射到 $\mathcal{B}$ 的同态函数。对于任意原子变量 $P$，指定 $V(P) \in \mathcal{B}$，并按如下规则递归扩展至复合公式：

    *   **否定/排斥** ($\neg$):
        $$ V(\neg(A)) = \neg^\mathfrak{M} V(A) $$

    *   **一致性算子** ($1$):
        $$ V(A \text{ Con } B) = \begin{cases} \top & \text{若 } \rho(V(A)) > \rho(V(B)) \\ \bot & \text{其他情况} \end{cases} $$

    *   **分化算子** ($2$):
        $$ V(A \text{ Ext } B) = \begin{cases} \top & \text{若 } \rho(V(A)) = 2 \text{ 且 } \rho(V(B)) = 2 \\ \bot & \text{其他情况} \end{cases} $$

##### 5.2 有效性定义
公式 $A$ 在模型 $\mathfrak{M}$ 中是**有效**的，记作 $\mathfrak{M} \vDash A$，当且仅当对于任意变量赋值，都有 $V(A) = \top$。

#### 6. 元理论性质 (Metatheoretic Properties)

本节证明系统 LCD 相对于模型 $\mathfrak{M}$ 的可靠性（Soundness）及系统的相容性（Consistency）。

##### 6.1 可靠性定理 (Soundness Theorem)
**定理**：若 $\vdash_{LCD} A$，则 $\mathfrak{M} \vDash A$。
**证明**：只需证明所有公理在 $\mathfrak{M}$ 中有效，且所有推导规则均保持有效性（即若前提赋值为 $\top$，结论赋值必为 $\top$）。

**1. 公理验证**
*   **反自反律** $\overline{\neg(A \text{ Con } A)}$:
    对于任意 $x \in \mathcal{B}$，$\rho(x) > \rho(x)$ 恒为假。故 $V(A \text{ Con } A) = \bot$。
    因此 $V(\neg(A \text{ Con } A)) = \neg^\mathfrak{M} \bot = \top$。公理有效。

*   **真值阶序律** $\overline{(A \text{ Con } A) \text{ Con } \neg(A \text{ Con } A)}$:
    设 $v$ 为 $\mathfrak{M}$ 中的任意赋值。
    由反自反律证明已知，对于任意 $A$，$v(A \text{ Con } A) = \bot$。
    根据语义定义，$\rho(v(A \text{ Con } A)) = \rho(\bot) = 3$。
    进而 $v(\neg(A \text{ Con } A)) = \neg^\mathfrak{M} \bot = \top$。
    根据语义定义，$\rho(v(\neg(A \text{ Con } A))) = \rho(\top) = 1$。
    考察公理整体形式：$v(\text{左式} \text{ Con } \text{右式})$。
    判定条件为：$\rho(\text{左式}) > \rho(\text{右式})$，即 $3 > 1$。
    该不等式在自然数序中恒成立，故结果恒为 $\top$。公理有效。

**2. 推导规则验证**

*   **I. 基础结构规则**
    *   **传递律**: 实数序关系 $>$ 具有传递性。若 $\rho(V(A)) > \rho(V(B))$ 且 $\rho(V(B)) > \rho(V(C))$，则 $\rho(V(A)) > \rho(V(C))$。保持有效。
    *   **非对称律**: 若 $\rho(V(A)) > \rho(V(B))$，则 $\rho(V(B)) \ngtr \rho(V(A))$。故 $V(B \text{ Con } A) = \bot$，进而 $V(\neg(B \text{ Con } A)) = \top$。保持有效。
    *   **双重否定消除**: 布尔代数性质 $\neg(\neg x) = x$。保持有效。

*   **II. 一致性迭代规则**
    我们需要证明 $V(A \text{ Con } B) = \top \iff V(A \text{ Con } (A \text{ Con } B)) = \top$。
    令 $x = V(A), y = V(B)$。
    *   $(\Rightarrow)$ 假设 $x \text{ Con } y = \top$，即 $\rho(x) > \rho(y)$。
        由于 $\rho$ 取值 $\{1,2,3\}$，且 $y$ 最小秩为 1，故 $\rho(x) \ge 2$。
        右式内部项 $V(A \text{ Con } B) = \top$。
        右式整体变为 $V(A \text{ Con } \top)$。
        检验条件：$\rho(x) > \rho(\top) \iff \rho(x) > 1$。
        由 $\rho(x) \ge 2$ 可知条件成立，结果为 $\top$。
    *   $(\Leftarrow)$ 假设 $x \text{ Con } y = \bot$，即 $\rho(x) \le \rho(y)$。
        右式内部项 $V(A \text{ Con } B) = \bot$。
        右式整体变为 $V(A \text{ Con } \bot)$。
        检验条件：$\rho(x) > \rho(\bot) \iff \rho(x) > 3$。
        因 $\rho$ 最大值为 3，此条件恒假，结果为 $\bot$。
    双向蕴含成立，规则保持有效性。

*   **III. 分化与排斥规则**
    *   **分化阻断** ($\frac{A \text{ Ext } B}{\neg(A \text{ Con } B)}$):
        前提真 $\implies \rho(V(A)) = \rho(V(B)) = 2$。
        此时 $\rho(V(A)) \ngtr \rho(V(B))$，故 $V(A \text{ Con } B) = \bot$，结论 $V(\neg(\dots)) = \top$。保持有效。
    *   **判定的确定性** ($\frac{A \text{ Con } \neg(B)}{\neg(A \text{ Ext } B)}$):
        前提真 $\implies \rho(V(A)) > \rho(\neg^\mathfrak{M} V(B))$。
        若 $V(B) \in \{\alpha, \beta\}$，则 $\neg^\mathfrak{M} V(B) \in \{\beta, \alpha\}$，秩均为 2。此时需 $\rho(V(A)) > 2 \implies V(A) = \bot$。
        结论需验证 $V(A \text{ Ext } B)$ 是否为 $\bot$。因 $V(A)=\bot$ 秩为 3 ($\neq 2$)，分化算子条件不满足，结果为 $\bot$，其否定为 $\top$。
        若 $V(B) \in \{\top, \bot\}$，则 $V(A \text{ Ext } B)$ 因 $B$ 秩不为 2 而恒为 $\bot$，结论恒真。保持有效。
    *   **分化对称性**: 等号对称性 $\rho(A)=\rho(B) \iff \rho(B)=\rho(A)$。保持有效。
	*   **分化显现律**:
	    $$ \frac{\neg(A \text{ Con } \neg(A)), \quad \neg(\neg(A) \text{ Con } A)}{A \text{ Ext } \neg(A)} $$
	    设前提在赋值 $v$ 下均为 $\top$。令 $x = v(A)$，则 $v(\neg(A)) = \neg^\mathfrak{M} x$。
	    
	    *   前提 1 为真 $\implies v(\neg(A \text{ Con } \neg(A))) = \top \implies v(A \text{ Con } \neg(A)) = \bot$。
	        根据算子 $1$ 定义，这意味着 $\rho(x) \ngtr \rho(\neg^\mathfrak{M} x)$。
	    *   前提 2 为真 $\implies v(\neg(\neg(A) \text{ Con } A)) = \top \implies v(\neg(A) \text{ Con } A) = \bot$。
	        根据算子 $1$ 定义，这意味着 $\rho(\neg^\mathfrak{M} x) \ngtr \rho(x)$。
	    
	    在模型 $\mathcal{B}$ 中，$\rho$ 的值域为 $\{1, 2, 3\}$。考察元素 $x$ 及其补元 $\neg^\mathfrak{M} x$ 的可能秩组合：
	    1.  若 $x = \top$，则 $\rho(x)=1, \rho(\neg^\mathfrak{M} x)=3$。满足 $1 > 3$，违背前提 1。
	    2.  若 $x = \bot$，则 $\rho(x)=3, \rho(\neg^\mathfrak{M} x)=1$。满足 $1 > 3$，违背前提 2。
	    3.  若 $x \in \{\alpha, \beta\}$，则 $\rho(x)=2, \rho(\neg^\mathfrak{M} x)=2$。
	        $2 > 2$ 不成立，满足前提 1。
	        $2 > 2$ 不成立，满足前提 2。
	    
	    因此，前提成立当且仅当 $\rho(x) = 2$ 且 $\rho(\neg^\mathfrak{M} x) = 2$。
	    
	    考察结论 $A \text{ Ext } \neg(A)$：
	    算子 $2$ 的真值条件为：左参数秩为 2 且 右参数秩为 2。
	    已知 $\rho(v(A))=2$ 且 $\rho(v(\neg(A)))=2$，条件满足。
	    故 $v(A \text{ Ext } \neg(A)) = \top$。规则保持有效性。

*   **IV. 混合算子扩张规则**
    *   **分化的否定无关性**:
        注意到对于任意 $x \in \mathcal{B}$，$\rho(x)=2 \iff \rho(\neg^\mathfrak{M} x)=2$（$\alpha, \beta$ 互补且同秩）。
        因此 $V(B \text{ Ext } C) = V(B \text{ Ext } \neg(C))$ 恒成立。
        前提与结论完全等价。保持有效。
    *   **独立前件引入**:
        前提1: $\rho(V(A)) > \rho(V(B \text{ Con } C))$。因算子 $1$ 输出仅为 $\top, \bot$，若输出 $\bot$ ($\rho=3$) 则前提1不可能成立（无元素秩 $>3$）。故必须 $V(B \text{ Con } C) = \top$ 且 $\rho(V(A)) > 1$（即 $\rho(V(A)) \ge 2$）。
        前提2: $V(A \text{ Ext } D) = \top \implies \rho(V(A)) = 2$ 且 $\rho(V(D)) = 2$。
        结合得：$\rho(V(A)) = 2$。
        结论: $A \text{ Con } (D \text{ Con } (B \text{ Con } C))$。
        代入已知值：$D \text{ Con } (B \text{ Con } C) \to D \text{ Con } \top$。
        因 $\rho(V(D))=2 > 1=\rho(\top)$，故 $D \text{ Con } \top = \top$。
        结论化简为 $A \text{ Con } \top$。
        因 $\rho(V(A))=2 > 1$，结果为 $\top$。保持有效。
    *   **元分化转换 I & II**:
        前提包含子式 $(A \text{ Con } B) \text{ Ext } C$。
        定义中 $1$ 算子输出值域为 $\{\top, \bot\}$，其秩分别为 $1, 3$。
        定义中 $2$ 算子要求左侧参数秩为 $2$。
        因此，$V((A \text{ Con } B) \text{ Ext } C)$ 恒为 $\bot$。
        前提恒假，规则在逻辑上真空满足，保持有效性。
    *   **弱链阻断**:
        假设前提在赋值 $V$ 下均为 $\top$。
        由前提 $B \text{ Ext } C$ 为真，根据定义可知 $\rho(V(B)) = 2$ 且 $\rho(V(C)) = 2$。
        由前提 $\neg(A \text{ Con } B)$ 为真，可知 $V(A \text{ Con } B) = \bot$，这意味着 $\rho(V(A)) \ngtr \rho(V(B))$。代入 $V(B)$ 的秩，得 $\rho(V(A)) \le 2$。
        
        考察结论中的内层子式 $C \text{ Con } B$：
        因 $\rho(V(C)) = 2$ 且 $\rho(V(B)) = 2$，不等式 $2 > 2$ 不成立，故 $V(C \text{ Con } B) = \bot$。
        
        考察结论的外层结构 $A \text{ Con } (C \text{ Con } B)$：
        将其转化为 $A \text{ Con } \bot$。根据算子 $\text{ Con }$ 的定义，此式为真的条件是 $\rho(V(A)) > \rho(\bot)$。
        已知 $\rho(\bot) = 3$，且模型中秩的最大值为 3，因此 $\rho(V(A)) > 3$ 恒假。
        故 $V(A \text{ Con } (C \text{ Con } B)) = \bot$。
        
        最终，结论 $\neg(A \text{ Con } (C \text{ Con } B))$ 的真值为 $\neg^\mathfrak{M} \bot = \top$。规则保持有效性。
##### 6.2 系统的相容性与非平凡性 (Consistency and Non-triviality)

**定理 (一致性)**：系统 LCD 是一致的，即 $\nvdash_{LCD} A \land \neg(A)$。
**证明**：
根据 6.1 节的可靠性定理，若 $\vdash_{LCD} \phi$，则 $\mathfrak{M} \vDash \phi$。
假设系统不一致，则存在公式 $P$ 使得 $\vdash P$ 且 $\vdash \neg(P)$。
这意味着 $\mathfrak{M} \vDash P$ 且 $\mathfrak{M} \vDash \neg(P)$。
即对于任意赋值，$⟦ P ⟧ = \top$ 且 $\neg^\mathfrak{M} ⟦ P ⟧ = \top$。
这在布尔代数 $\mathcal{B}$ 中是不可能的（$\top \ne \bot$）。
故系统一致。

**定理 (非平凡性)**：系统 LCD 并非所有合式公式都是定理。
**证明**：
考察公式 $\phi = \alpha \text{ Con } \beta$（其中 $\alpha, \beta$ 为变量）。
构造赋值 $v$ 使得 $v(\alpha) = \alpha, v(\beta) = \beta$。
$\rho(\alpha)=2, \rho(\beta)=2$。
$\rho(\alpha) \ngtr \rho(\beta)$，故 $V(\alpha \text{ Con } \beta) = \bot$。
因为存在使得 $\phi$ 为假的赋值，故 $\phi$ 不是定理。
同理，$\psi = \alpha \text{ Ext } \top$ 在模型中也恒为假。
因此，系统 LCD 既包含定理（如公理），也包含非定理，是非平凡的。

#### 7. 意向模型与形式语义诠释 (Intended Model and Formal Interpretation)

上述有限秩布尔代数模型（第5节）仅用于确立系统的纯形式相容性。系统 LCD 的设计初衷是为了捕捉数理逻辑中关于**一致性强度（Consistency Strength）**、**不可判定性（Undecidability）** 以及**力迫法（Forcing）** 扩张的核心直觉。

在本节中，我们将建立一个非形式化的意向模型 $\mathfrak{I}$，将 LCD 系统的符号映射到具体的元数学概念上。我们主要关注由公理 (A1), (A2) 以及推导规则 (R1.1)-(R4.5) 构成的子系统。我们不得不把这个模型的形式化作为开放问题保留给读者。

##### 7.1 语义映射

在意向模型 $\mathfrak{I}$ 中：
*   **变量 $A, B, \dots$**：代表一阶递归公理化理论（如 PA, ZFC, ZFC+CH 等）。
*   **算子 $\text{ Con }$ (一致性序)**：$A \text{ Con } B$ 解释为“理论 $A$ 的一致性强度严格高于理论 $B$”，或者更具体地，“$A$ 能够在元数学上证明 $\text{ Con }(B)$（即 $A \vdash \text{ Con }(B)$）”。这对应于解释性（Interpretability）概念。
*   **算子 $\text{ Ext }$ (独立性分支)**：$A \text{ Ext } B$ 解释为“命题 $B$（或将其视为公理的单句理论）对于理论 $A$ 是独立的”。即 $A \nvdash B$ 且 $A \nvdash \neg B$，且 $A + B$ 是 $A$ 的一个相容扩张（类似于通过力迫法添加一个类属过滤器）。
*   **算子 $\neg$ (元逻辑否定)**：$\neg(A)$ 解释为“理论 $A$ 是不一致的”，或“$A$ 包含逻辑矛盾”。

##### 7.2 公理模式的诠释

*   **A1. 反自反律 ($\overline{\neg(A \text{ Con } A)}$)**
    *   **诠释**：这是**哥德尔第二不完备性定理**的直接体现。任何包含一定算术强度的相容理论 $A$ 都无法在系统内证明自身的一致性（即无法构造自身的模型）。因此，“$A$ 证明 $A$ 的一致性”这一命题在元逻辑上被判定为假，其否定（不一致性标记）为真。

*   **A2. 真值阶序律 ($\overline{(A \text{ Con } A) \text{ Con } \neg(A \text{ Con } A)}$)**
    *   **诠释**：这体现了**Ex Falso Quodlibet**（由假得全）的逻辑原则。$(A \text{ Con } A)$ 是一个必然假的陈述（逻辑矛盾）。一个不一致的系统（或基于矛盾的前提）拥有平凡的“爆炸”解释力，它可以“解释”或推导出任何结论，包括绝对真理 $\neg(A \text{ Con } A)$。

##### 7.3 推导规则的元数学对应

**I. 基础结构 (Consistency Hierarchy)**
*   **R1.1 (传递律)**：一致性强度构成一个偏序集。若 ZFC 强于 PA，PA 强于 PRA，则 ZFC 强于 PRA。
*   **R1.2 (非对称律)**：一致性阶梯是严格良基的，不存在两个理论互为更强的一致性证明者（排除循环解释）。
*   **R1.3 (双重否定消除)**：经典逻辑在元层面的有效性，“不一致是不可能的”等同于“一致”。

**II. 一致性迭代 (Provability Logic)**
*   **R2.1 & R2.2 (迭代与塌缩)**：这反映了算术化的一致性陈述的性质。如果 $A$ 证明了 $B$ 的一致性，这是一个算术事实（$\Sigma_1^0$ 语句）。在一个足够强的理论 $A$ 中，能够验证“我证明了 $B$ 的一致性”。这对应于可证明性逻辑中的 $\Box P \leftrightarrow \Box \Box P$ 原则。

**III. 分化与排斥 (Independence & Forcing)**
*   **R3.2 (判定的确定性)**：如果理论 $A$ 能够证明 $B$ 是不一致的（即 $B$ 在 $A$ 中被证伪），那么 $B$ 就不能作为 $A$ 的独立扩张分支。独立性的前提必须是相容性。
*   **R3.4 (分化显现律 / The Rosser Rule)**
    *   前提：$\neg(A \text{ Con } \neg(A))$ —— $A$ 不能证明自身是不一致的（即 $A$ 是相容的）。
    *   前提：$\neg(\neg(A) \text{ Con } A)$ —— “$A$ 不一致”这一事实并不蕴含 $A$（防止平凡情况）。
    *   结论：$A \text{ Ext } \neg(A)$。
    *   **诠释**：这是对**罗塞尔句子（Rosser Sentence）** 或不完备性定理推论的刻画。如果 $A$ 是相容的，那么命题“$A$ 是不一致的”（$\neg \text{ Con }(A)$）在 $A$ 中是不可判定的。因此，我们可以在 $A$ 之上构建一个非标准模型，在该模型中 $A$ 成立但 $\neg \text{ Con }(A)$ 也成立。即“自身的不一致性”成为了一个合法的逻辑分支选项。

**IV. 混合算子扩张规则 (Structural Preservation & Robustness)**

*   **R4.1 (分化的否定无关性)**：
    *   **直觉**：这是**独立性的对称性**。在力迫法或模型扩张中，如果命题 $C$ 对于理论 $B$ 是独立的（即 $B$ 无法决定 $C$），那么 $B$ 对 $C$ 的否定 $\neg C$ 同样无法决定。对于元理论 $A$ 而言，观察到 $B$ 在 $C$ 处发生分支，等价于观察到 $B$ 在 $\neg C$ 处发生分支。独立性是一种关于可判定性的结构属性，不依赖于具体的真值极性。

*   **R4.2 (独立前件引入)**：
    *   **直觉**：这体现了**由弱到强的逻辑扩张的保守性（Conservativity）**。如果理论 $A$ 已经确立了“$B$ 强于 $C$（$B \vdash \text{ Con }(C)$）”这一元数学事实，且 $D$ 是 $A$ 的一个独立扩张分支（$A \text{ Ext } D$），那么这个既定的一致性关系在扩张后的语境中依然保持可见。换言之，一致性强度的证据具有**绝对性（Absoluteness）**，它不会因为引入了新的独立公理 $D$ 而消失或变得不可通过 $D$ 访问。

*   **R4.3 & R4.4 (元分化转换 I & II)**：
    *   **直觉**：这两条规则的核心在于**一致性证据的鲁棒性（Robustness）**。
        *   **R4.3** 表明，如果 $A$ 对 $B$ 的解释力（$A \text{ Con } B$）能够经受住外部命题 $C$ 的干扰（即与 $C$ 的分化相容），说明 $A$ 具有足够高的元逻辑地位，能够将 $C$ 纳入其框架内，使其成为 $B$ 的潜在解释语境。
        *   **R4.4** 则从反面划定了界限：这种鲁棒性意味着 $A$ 的主导地位是刚性的，它排除了那些会导致逻辑层级混乱的病态依赖。如果一致性关系在 $C$ 的干扰下依然成立，那么 $A$ 就能够断言：绝不允许通过否定 $C$ 来构建反向的、破坏既定层级的解释路径。简言之，**真理的阶序不会因侧向的扰动而崩塌**。

*   **R4.5 (弱链阻断)**：
    *   **直觉**：这描述了一种“**信任链断裂**”的**传递性**。
        前提一 $\neg(A \text{ Con } B)$ 指出 $A$ 自身太弱，无法为 $B$ 提供担保；
        前提二 $B \text{ Ext } C$ 指出 $C$ 仅仅是 $B$ 视野之外的一个独立命题（侧向扩张），而非更强的公理系统。
        结论指出：既然 $A$ 连 $B$ 本体都无法掌控，而 $C$ 又是 $B$ 所无法决定的，那么 $A$ 没有任何理由相信 $C$ 能充当 $B$ 的一致性担保人。独立性（$C$）不能凭空产生一致性强度，弱者（$A$）无法通过引用未知的第三方（$C$）来完成它本就无力完成的验证。
